The Master Sendmail.mc File

The authors of sendmail+IDA provide the Sendmail.mc file which contains the true ``guts'' of what becomes the sendmail.cf file. Periodically, new versions are released to fix bugs or add functionality without requiring a full release and re-compilation of sendmail from sources.

It is important not to edit this file.
