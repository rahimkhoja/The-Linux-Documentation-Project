Chapter 7. Unmaintained HOWTOs

There are a number of unmaintained documents, which can be found at: http://tldp.org/authors/unmaint.html.

These are kept around since old documentation is sometimes better than none. However, you should be aware that you are reading old documentation.

If the date tag is too old, please consider the HOWTO as unmaintained.

However, there are exceptions, such as those that document older programs and procedures that still might be in use, or those that cover more obscure topics.
