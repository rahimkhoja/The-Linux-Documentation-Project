1. Preface

Copyright and license information, as well as credits, are at the end of this document.

This HOWTO is maintained by Bryan Henderson, bryanh@giraffe-data.com. You can get the current version of this HOWTO from the Linux Documentation Project.

This document is mainly geared toward Linux 2.4. It covers secondarily earlier Linux. It covers newer Linux only partially, but the author's goal to cover it completely in a future release. See in particular Section 12. The subject changed quite a bit over the course of Linux 2.5 releases, and this document ignores 2.5, since the 2.5 releases were not considered production releases. Instead, the document treats all the changes in 2.5 releases as if they were new with Linux 2.6.

See Section 17 for a history of the document.
