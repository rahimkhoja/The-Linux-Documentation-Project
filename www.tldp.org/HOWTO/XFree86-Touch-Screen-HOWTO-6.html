6. Configuration under X.Org

(from Kenan Bektas)

Example xorg.conf for an Elo TouchSystems 2216 AccuTouch® USB Touch-Monitor: 

Section "InputDevice"
    Identifier "elo"
    Driver "elo"
    Option "AlwaysCore"
    Option "MinX"          "300"
    Option "MaxX"          "3700"
    Option "MinY"          "300"
    Option "MaxY"          "3700"
    Option "ScreenNumber"  "0"
    Option "ButtonNumber"  "1"
EndSection
