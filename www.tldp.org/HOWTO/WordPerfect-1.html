1. Introduction

1.1 Overview

The purpose of this document is to discuss WordPerfect on Linux. Discussed are running WP for SCO, WP 6.0 from Caldera, and a discussion on the WordPerfect 7.0 beta.

This document is an update to the previous mini-howto that described notes on setting up WordPerfect 6.0 for SCO UNIX (ODT) on Linux.

This document is not endorsed nor sponsored by Corel Corporation or Software Development Corporation.

1.2 History

  • Version 1.0 -- Description on how to use WordPerfect 5.1 and 6.0 for SCO ODT 3.0 on Linux
  • Version 2.0 -- Update and inclusion of information on Caldera's WordPerfect 6.0 distribution and the WordPerfect 7.0 beta.
