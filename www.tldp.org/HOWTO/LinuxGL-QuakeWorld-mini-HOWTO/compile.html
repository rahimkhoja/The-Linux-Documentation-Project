6. Compile!

Compile (finally!)

Just run make with no parameters from where the Makefile is located: 
	cd q1src/QW
	make

If all goes well, you will have a binary, glqwcl.glx, in the releasei386-glibc directory. Copy glqwcl.glx to another directory (like "qw") and then from a Quake cd or other quake installation, copy the id1 directory and any other game directories you like/have - ctf, Fortress etc - to your new quakeworld directory.
