5. Software

There are some programs available, that can automate the process of creating CDs from MP3 files. Here is arbitrarily selected list:

  • burnmp3 - Program to automate burning with DAO method. http://richardsnow.bizland.com/burnmp3/.

  • mp32dao - a script from cdrdao distribution, in the contrib directory. http://cdrdao.sourceforge.net/.
