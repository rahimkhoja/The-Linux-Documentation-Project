Using a Direct Connection

Assume you use a direct line connecting your system vstout to tiny. Very much like in the modem case, you have to write a system entry in the sys file. The port command identifies the serial port tiny is hooked up to.

In the port file, you have to describe the serial port for the direct connection. A dialer entry is not needed, because there's no need for dialing.
