2.2. Uninstalling VLS

2.2.1. Windows

Go to the Control Panel, click on Add and remove programs, select VLS and click on Modify/Remove and follow the steps to uninstall the program .

2.2.2. If you compiled VLS from sources

Go to the directory containing VLS sources and run :

# make uninstall

Then you can remove the VLS sources .
