1. LDP guides

When talking about TLDP, people often forget about the Guides section. Check out our collection of the longer, in-depth documents!

2. New documents

2.1. Fresh@LDP

There were no 1.0s announced by Greg this week, however our Spanish colleagues added a document titled Más Allá de Linux From Scratch, which is a continuation of the Linux From Scratch guide, including support for X, sound, printers and more.

2.2. Proposed new documents

  • Guylhem Aznar is thinking about a FAQ on Using the keypad under X and has requested information on certain problems that people seem to face.

3. Updates

3.1. HOWTOs

We received updates for the following HOWTOs and mini-HOWTOs:

  • The Connecting X Terminals to Linux Mini-HOWTO v1.2 by Salvador J. Peralta

  • David Horton sent in v1.1 of the Pocket Linux Guide.

  • A dot-oh, but it's v2.0 of the Linux 2.4 SCSI subsystem HOWTO, by Douglas Gilbert.

  • Svetoslav Slavtchev is still polishing the XFree Local Multi-User HOWTO, remember, the new HOWTO covering the connection of multiple monitors, soundcards, keyboards etc. to one server.

4. Happenings in the LDP world

4.1. Online editor for DocBook

Stein Gjoen posted a URL to his LDP HOWTO Generator. Nicolas Chauvat however, said that doing something similar in PHP would be re-inventing the wheel and pointed the list to demo.plone.org and the Archetypes documentation to get started.

Elsewhere, David Merrill posted an update to wt2db (WikiText to DB) and announced that he was getting back to work on Lampadas. Nicolas Chauvat would be updating him with the things that have moved forward since David was last involved.

4.1. DFSG analysis of default LDP license

Since many of the mini-HOWTOs are under the default LDP license, Branden Robinson subjected the default LDP license to a Debian Free Software Guidelines analysis and proposed several changes to it. People further discussed the license and the changes on the the debian-legal mailing list.
