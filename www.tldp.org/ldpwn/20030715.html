1. New documents

1.1. Proposed new documents

  • Kian Spongsveen proposed to write a Sybase ASE / SQL Server HOWTO which would contain installation and the first steps of basic usage. Real-world known problems one could hit during installation and pointers to further information would also be a part of the HOWTO.
  • Ariel Mella proposed to write a HOWTO on integrating Linux services and MS Domains.

3. Happenings in the LDP world

3.1. TLDP-XSL stylesheets updated

David Horton sent in an update on the TLDP-XSL that he has been working on. His work is available as tldp-xsl-10JUL2003.tar.gz. The major changes since the last update includes producing formatting objects that can be converted into other formats using an FO processor.

4. Feedback and Contributions

Please help us create this weekly newsletter. Seen a site putting in a story with a link to the LDP? Completed the translation of any HOWTO? Know of any articles about the LDP published in newspapers or magazines? You can update us by sending in links and information at <feedback@en.tldp.org>.
