1. New document proposals

  • Ivan Baldinotti proposes a mini-HOWTO titled From endnote to LyX, which details migration from Endnote or MSWord to LyX. There were recent discussions on whether or not to add a note on LyX to the Author Guide.
  • Machtelt Garrels proposes to write detailed documentation for the creation of presentations using DocBook XML. Though some say this is overkill, she defends the proposal saying that she is looking for two important things: text-mode creation and compatibility with MS Windows users without having to fall back to HTML.

Join our discuss mailinglist or mail these people directly to let them know what you think!

2. Updated HOWTOs, FAQs and Guides

  • v2.3 of the Medicine HOWTO is now available. Gerardp Arnaez and Walt Pennington updated the existing links and added new ones to more websites dealing with the subject of Free/Open medical software. They try out most of the software they discuss, and talk about their experiences, difficulties they encountered and so on.
  • "Misko's triplet" has been updated again. This includes the FBB Packet-Radio BBS mini-HOWTO, the Linux+WindowsNT mini-HOWTO and the LILO mini-HOWTO.
  • There were a few typos in Daniel Nofftz's Athlon Powersaving HOWTO, these have been corrected in v1.1.2.
  • The Postfix-Cyrus-Web-cyradm-HOWTO guides you through the installation of the Postfix mail transportation agent and the Cyrus IMAP server. Web-cyradm is a graphical web-based user-administration interface to the system; data is stored in a MySQL database. Luc de Louw has a whole chapter about virus- and spam protection. Minor additions and corrections in v1.2.6 of this document.
  • It was argued on the discuss mailinglist that the Author Guide did not fully reflect TLDP procedures, so Emma Jane Hogbin submitted an Author Guide update (v4.2).
  • Eric S. Raymond has done some rephrasing in v2.9 of the The Unix and Internet Fundamentals HOWTO, which explains what your PC does and how it is done in non-technical language.

3. News in The LDP world

  • Removal of the Linux Cookbook:

    Michael Stutz let us know that he thinks the Linux Cookbook contains too many errors, bugs and licensing issues. Therefor, as the maintainer of the document, he asked us to remove the Cookbook until he has had time to fix the issues.

  • Anti-spam measures on the LDP mailing lists:

    Sergiusz Pawlowicz announced the start of a test-phase after installing the spamassassin filter on the submit mailinglist on Sunday morning - yes, we work while you sleep/party/have your Sunday-morning soft-boiled egg ;-)

    Sergiusz argued before that everybody should have their own E-mail filtering because TLDP didn't have the resources to do it on the server-side, but apparently he has found the hardware to do it.

    We hope that if the tests go well, the feedback list will also be filtered.

4. HOWTO contribute to TLDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
