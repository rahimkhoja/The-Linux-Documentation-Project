1. Documents submitted for review

  • Kai Schlachter polished up the documentation he wrote about the installation process of glibc, took a crash course in docbook and sent in his contribution, which he would like to call the Glibc mini-HOWTO. It's not complete yet: Kai still needs to fine tune this document to be compliant with the newer versions of glibc, but comments on the document are welcome.
  • Lars Strand wrote an OLSR (Optimized Link State Routing Protocol) IPv6 HOWTO (source XML + images), describing this routing protocol for mobile ad-hoc networks. The document has had a technical review, the result of which is now here. Lars is of the opinion that a language review would be valuable.

2. New document proposals

  • Michael Schlenstedt is working on an ISDN HOWTO (Table of contents) (German version), explaining ISDN-support in 2.6 kernels. In the new kernels, a lot has changed and Michael wants to describe installation and give some background information about CAPI and ISDN4Linux so as to be able to troubleshoot.
  • Luigi Bitoni proposed a text documenting the combination of Apache, Tomcat, Sun's JDK and mod_jk. Luc De Louw (author of a slightly related MMBase Installation HOWTO), suggested combining the efforts, because he is not a Tomcat and mod_jk specialist.

3. Updated HOWTOs, FAQs and Guides

  • Mendel Cooper has bug fixes and more examples in the Advanced Bash Scripting Guide v2.5.
  • Major points of attention in v0.7 of the LVM HOWTO are the inclusion of a discussion about LVM 2 and the addition of a FAQ section. Author AJ Lewis handed over the copyright to RedHat.

4. News in The LDP world

4.1. Iceland Icenet mirror moved

Albert Ragnarsson let us know that the TLDP mirror over at http://www.eldveggir.com/eldveggur/LDP/ no longer exists because he dropped the domain. A replacement is available from http://ldp.icenet.is/. This mirror is updated daily.

5. HOWTO contribute to TLDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
