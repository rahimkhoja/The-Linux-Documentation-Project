1. New documents @ TLDP

  • Adrian Offerman surprises us with v1.0 of the MMBase Installation HOWTO, which describes the installation of this content management system on RedHat Linux using Tomcat as an application server. He also provides info on how to integrate it with MySQL and Apache.

2. Documents submitted for review

  • Steve Peace submitted a document called the Qmail-scanner+Clamav HOWTO, detailing the installation of the ClamAV GPL'ed anti-virus toolkit on a Qmail server. Steve awaits your comments on this initial draft.
  • Lars Strand updated his Mobile IPv6 HOWTO with new images, two new sections and error fixes.
  • Peter J. Salzman submitted v1.0.1 of the Linux Gamer's HOWTO, LG HOWTO for short.
  • David Jao is looking for reviewers for his DVD Playback HOWTO. The document describes how to view DVD movies on a Linux computer with a DVD drive and also helps troubleshoot common problems.

3. New document proposals

  • Lars Strand wrote an Optimized Link State Routing Protocol (OLSR) IPv6 HOWTO for Linux, and would like to submit it to the LDP for inclusion in the collection. Comments on the above document are welcome.
  • David Lawyer suggested that a one-page intro to LinuxDoc might be a good idea. Machtelt Garrels offered to write a DocBook counterpart.

4. Updated HOWTOs, FAQs and Guides

  • Eric S. Raymond's Linux Reading List HOWTO has reached v3.0. Dot-zero always indicates a major update. From this one all out-of-date books have been removed and a couple of new ones have been added.
  • V2.22 of Daniel Quintiliani's Windows Newsreaders under Linux HOWTO contains minor corrections.
  • The Introduction to Linux guide by Machtelt Garrels has an ISB Number and a few formatting changes. The document is being translated to Hindi by Alok Kumar.

5. News in The LDP world

5.1. Acer Laptop HOWTO removed

Harish Tampi has requested the removal of the Acer Laptop HOWTO due to a lack of time from his part to keep it up to date. However, info for this brand of portables can be found at the Linux on Laptops site, section Acer Laptop and Notebook Computers, or at Tuxmobile, which hosts a similar page.

5.2. French LDP effort

You might have noticed that the French translation effort, http://fr.tldp.org/, has been pretty motionless for a long time. However, the Traduc site is quite active.

Apart from regular news updates, this sites features the traduc po, which allows tracking of the state of translations for different packages. Also worth noticing is the translation to French of the Linux Gazette.

6. HOWTO contribute to TLDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
