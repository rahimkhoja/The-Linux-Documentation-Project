2. Updates

2.1. HOWTOs

We're getting in full speed again, be prepared!

  • Saqib Ali provided us with v4.1.0 of his Apache based WebDAV Server with LDAP and SSL HOWTO. If you were wondering: WebDAV stands for Web enabled Distributed Authoring and Versioning. Saqib's objective is to set up an Apache + MySQL + PHP + WebDAV based application server, using LDAP for authentication.
  • If you feel like playing a bit with your kernel, here's v1.04 of Bryan Henderson's Linux Loadable Kernel Module HOWTO.
  • We've got a lot of updates from the VideoLAN project. VideoLAN is a complete software solution for streaming MPEG video on high-bandwidth networks. The project consists of VLS, the VideoLAN server and VLC, the client, which can accidentally also serve videos.

    The documentation consists of the following:

    • The VideoLAN HOWTO
    • The VideoLAN Quick-start
    • The VLC User Guide
    • The VLS User guide

    Many thanks to Alexis de Lattre, Marc Ariberti, Anil Daoud, Clement Stenac, Henri Fallon, Johan Bilien, Mathieu Gautier and all the others volunteering for this project!

  • And Svetoslav Slavtchev mailed us v1.1.1 of his XFree Local Multi-User HOWTO, which explains how to use the Backstreet Ruby kernel for supporting multiple keyboards, mice and monitors on one PC.
  • If you want to try an alternative form of wireless connection, check out v1.24 of Miroslav "Misko" Skoric's FBB Packet-radio BBS mini-HOWTO, covering the installation and use of the popular amateur packet-radio BBS server software "FBB".
  • v3.13 of Miroslav "Misko" Skoric's LILO mini-HOWTO.
  • The Linux+WindowsNT mini-HOWTO v2.13. from Miroslav "Misko" Skoric.
  • Daniel Quintiliani sent in v1.1 of the Xnews under Linux HOWTO, describing how to set up and use the Windows client Xnews running Wine emulator.
