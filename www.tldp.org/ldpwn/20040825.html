1. New document proposals

  • Diego M. Vadell - UYR send a mail to the discuss list asking The LDP to consider the inclusion of his Compiling-HOWTO in The LDP collection. This howto wants to answer the question "How to compile", and also give an lightweight explanation of how does an ELF binary works, how does the shared libraries are found and who are the actors of the compiling process, in order to give a system administrator the tools and knowledge to troubleshoot compiling and general library problems. Comments on his HOWTO are welcome.
  • Jose Maria Lopez's team at bgSEC wrote a fairly comprehensive Physical Security HOWTO (Compressed with bzip2 for downloading). The document would be included in the Spanish LDP collection and Jose Maria Lopez would like to see the document translated into English and included with the English LDP collection. Please contact the team if you would like to help with the translation.

2. Updated HOWTOs, FAQs and Guides

  • If every minute is important for you, the Coffee Making HOWTO is just the thing. Steadily heading for 1.0, v0.90 introduces a name change and info on Coffee+LAMP, apart from other minor changes.
  • Svetoslav Slavtchev's XFree Local Multi-User HOWTO has info about the Mandrake/Debian IsolateDevice for configuring the X server to be used by multiple local users.
  • Bin Nguyen sent in v0.65 of the Linux Filesystem Hierarchy, in which he tells a bit more about himself and adds two new sections, a.o. one for providing feedback. Also by Binh: the Linux Dictionary, v0.16.
  • A.J. Lewis has additional info about the boot and root filesystems when using LVM. Apart from that, minor fixes and updates in v0.13 of the LVM HOWTO.
  • Minor bug fixes in v1.4 of David Braun's Disk Encryption HOWTO.
  • David S. Lawyer updated his Plug and Play HOWTO. It now covers /proc/isapnp and sysfs in the 2.6 kernels. Some info about hot pluggable devices and USB has been added as well.
  • Minor corrections in the Linux MP3 CD Burning mini-HOWTO v1.11, submitted by Greg Wierzchowski.

4. HOWTO contribute to The LDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
