2. New documents

2.1. Fresh@LDP

  • Claudio Erba, Chris Karakas, Andre Purfield and Fortunato Matarazzo wrote a PHP-Nuke: Management and Programming HOWTO.

    There has always been the necessity to have a definitive guide on PHP-Nuke. This tutorial describes the installation and structure of PHP-Nuke and the details of customizing the front end to suit the users' needs. The architecture of PHP-Nuke, with its modules, blocks, topics and themes is presented in detail, as well as the interplay of PHP and MySQL for the creation of a mighty content management system (CMS).It also delves into more advanced issues, like the programming of PHP-Nuke blocks and modules.

  • Ian Ward's Debian and Windows Shared Printing mini-HOWTO has been released

    This HOWTO will demonstrate how to use command-line tools to configure your Debian system for printing. It will explain how to send documents from Linux to Windows printers and how to share Linux printers with Windows PCs. Some troubleshooting examples are also given.

  • Mark Hubbard's VB6 To Tcl mini-HOWTO has been added to the LDP collection

    A 15 Minute Tcl Tour For Visual Basic and VBScript Programmers.

2.2. Proposed new documents

  • Chris Albert proposed to write an Anti-Spam-HOWTO
  • Gerardo Arnaez would like to write a HOWTO on qmail, vpopmail, qmailadmin server (on Debian) that allows stmp after POP authentication.

2.3. Submitted for review

  • Jorje Bernal submitted a Linux Coloring mini-HOWTO.

    Mini-HOWTO to use the ansi sequences to put colors in the shell.

  • Gerardo Arnaez submitted a Qmail, Vpopmail, Qmailadmin HOWTO for review.

    A cookbook to set up a DEBIAN based Qmail servers that allows virtual emails and SMTP after POP authentication.

  • Howard Shane submitted a Scanner-HOWTO for review.

    This document was written to assist the linux user in setting up their raster image scanner device, including flatbed, hand-held, video- and still-cameras, frame-grabbers and so on. It includes information on interface types, kernel support, making and accessing the necessary devices, and finally brief descriptions of applications available for use both at the command line and within X-windows to access scanner hardware.

4. Happenings in the LDP world

4.1. Anti-Spam-HOWTO

Chris Albert asked if the discussion last year regarding writing an Anti-Spam-HOWTO got off ground and offered to help. Quite a few others were interested in helping. Greg Ferguson said that he has been following the discussion with interest and noted that collaborative authoring using CVS is the direction what he hopes the LDP would move.

Hal Burgiss announced that Chris had set up a mailing list for anyone who is interested in contributing, or just hanging out. To subscribe, send a message to:

<LISTSERV@LISTS.MCGILL.CA>

with a blank subject line , and body without signature:

SUB ANTI-SPAM-HOWTO firstname lastname

Chris is working on the list archives for the ANTI-SPAM-HOWTO list, but the list is already discussing the content of the HOWTO and an outline draft is in the CVS.
