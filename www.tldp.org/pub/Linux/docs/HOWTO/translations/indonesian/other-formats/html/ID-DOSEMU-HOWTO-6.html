6. dosemu dan Netware

6.1 Bagaimana saya dapat mengakses Netware dari dosemu?

Seperti biasanya, akses melalui file system Linux lebih sering dilakukan. Mount drive Netware anda dengan utilitas Netware Caldera atau Volker Lendecke's free ncpfs utility ( ftp://ftp.gwdg.de:/pub/linux/misc/ncpfs). Jika anda membutuhkan akes real IPX, misal menjalankan "syscon" Novell, baca ../doc/NOVELL-HOWTO.txt.
