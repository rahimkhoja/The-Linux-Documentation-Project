3.2 PCMCIA

  • PCMCIA
    http://hyper.stanford.edu/HyperNews/get/pcmcia/home.html

Driver PCMCIA kebanyakan mendukung semua kontroler PCMCIA, termasuk Databook TCIC/2, Intel i82365SL, Cirrus PD67xx, dan Vadem VG-468 chipsets. Motorola 6AHC05GA controller yang digunakan dalam Hyundai tidak didukung. Lihat lampiran B untuk daftar card PCMCIA yang didukung.
