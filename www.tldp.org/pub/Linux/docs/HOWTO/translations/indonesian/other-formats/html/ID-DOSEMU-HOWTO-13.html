13.2 Saya ingin membantu. Siapa yang harus saya hubungi?

Proyek dosemu adalah usaha sebuah tim. Jika anda ingin memberi kontribusi, lihat DPR (dosemu Project Registry). Copynya dapat ditemukan dalam ../doc/DANG (97/2/9).
