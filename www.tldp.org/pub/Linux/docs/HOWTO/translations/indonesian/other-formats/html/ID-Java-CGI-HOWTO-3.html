3. Mensetup Server Anda untuk Menjalankan Program Java CGI (Bentuk Ringkas)

  • Ambillah paket Java CGI dari Java CGI. (Nomor versi mungkin berubah).
  • Bukalah distribusi dengan perintah : 
         gzip -dc java_cgi-0.4.tgz | tar -xvf -
    (Jika nomor versi berubah, selanjutnya gunakan instruksi-instruksi dari distribusi).
  • Editlah Makefile yang anda temukan di direktori java_cgi-0.4 yang baru tercipta agar sesuai dengan sistem anda.
  • Sebagai root, jalankan make install. Hal ini akan mengkompilasi program Java, mengaplikasikan informasi spesifik sistem anda dan menginstal berbagai file. Jika anda ingin versi HTML dokumentasi ini dan sebuah dokumen percobaan, jalankan make all.
  • Anda siap untuk lanjut.
