Algunos servidores NTP conocidos

Una lista de servidores públicos puede obtenerse de: http://www.eecis.udel.edu/~mills/ntp/servers.html. Por favor lea la información de utilización de la página antes de utilizar un servidor. No todos los servidores tienen el suficiente ancho de banda para permitir un gran número de sistemas sincronizándose con ellos. Así que es buena idea contactar con el administrador de sistemas antes de utilizar su servicio NTP.
