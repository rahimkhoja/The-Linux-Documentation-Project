so I suspect that some kind of porting would be possible.

so the (semi-?) proprietariness doesn't worry me that much.

have a buzz page for hot news. It has been dead since 1997
which worries me more. I'll try to contact them to see if

in return showcase a NetHelp styled HOWTO.
