Does anybody work at, or have some relationship with, a printer? If we can
get a printer to put the design on die, we should be able to get good rates
on actual offset printing, because they only need to change the
name/email/phone for each person. This might be more efficient than having
individuals go to Kinko's, although of course anybody that wants to can do
that. If somebody is willing to take this to a printer, we could all order
them at the same time and have them mailed to us.

FYI, Kinko's (at least in my area!) doesn't do offset printing, only laser.
This is not too bad, but offset printing gives much better quality. They
accept offset orders, but send them out.
