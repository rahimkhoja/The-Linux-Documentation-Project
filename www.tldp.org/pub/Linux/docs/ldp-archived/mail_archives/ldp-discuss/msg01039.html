FYI; some recent changes done to the LDP site:

- cleaned up some of the "links" pages; added link to the "Linux
  Development Projects" page (www.linuxdoc.org/devel.html) from
  each of the  "links" pages. I could never find that page when
  I wanted to...not sure if others had the same problem. It contains
  some excellent references.

- created a "volunteers" list which take the place of the old "team"
  list (www.linuxdoc.org/vlist.html). Take notice of the substantial
  list of those that have contributed content to the LDP. If you're
  not on the list and you should be (and want to be), let me know.

- added white background to search results...improves the readability.

Josh is currently working on improving the readability of
the home page, and on a method of classification for the
content on the "Docs" (www.linuxdoc.org/docs.html) page.
