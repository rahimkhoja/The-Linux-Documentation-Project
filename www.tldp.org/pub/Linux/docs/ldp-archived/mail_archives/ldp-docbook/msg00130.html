My main problem with sgedit is that although you can create a simple
style-sheet for your document, the program doesn't quite obey the rules
in the style-sheet when displaying the file (it doesn't put a line-break
before or after tags, no matter the settings).
A TCL programmer could probably fix it (the bug is somewhere in
sgeditTemplates.tcl), then we could provide a usable style-sheet for
authors.
