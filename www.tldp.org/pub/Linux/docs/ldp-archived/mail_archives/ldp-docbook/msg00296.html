On Oct 25,  7:04pm, Kai Blin wrote:
> Subject: sgml aequivalent of html <br>
> Hi folks,
>
> is there a sgml aeqivalent to the html <br> (that's line break) tag?
>
> I can't seem to find it. And no, <para> won't do it :)

Depends on how/where you want to use it. There is a <SBR>,
which is primarly used within <CmdSynopsis>. If you
want something with line-breaks, you might be better off
using <LiteralLayout>.

  SBR:           http://docbook.org/tdg/html/sbr.html
  CmdSynopsis:   http://docbook.org/tdg/html/cmdsynopsis.html
  LiteralLayout: http://docbook.org/tdg/html/literallayout.html

Someone else may be able to provide more information on this.
