The LinuxFocus Tip

Removing the damm trash can from KDE

In KDE 3 it is now possible to remove the "Trash" from the Desktop!

  1. Delete the ~/KDesktop/Trash directory
  2. Set KDE Control Center->Look & Feel->Desktop->Paths->Trash to /dev/null and see complain that you don't own it.
  3. Restart KDE.
Trash won't work because you don't own /dev/null.
Enjoy a wonderful clean desktop ;-)
