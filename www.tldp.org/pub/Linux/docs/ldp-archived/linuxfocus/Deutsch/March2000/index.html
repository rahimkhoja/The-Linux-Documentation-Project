LinuxFocus März 2000 Ausgabe


Willkommen zur März 2000 Ausgabe von LinuxFocus!
Die Welt um Linux erregt immer mehr das Interesse großer Unternehmen und allgemein von Leuten, die gerne dicke Geschäfte machen. Wir waren auch versucht, ein bißchen Einkommen durch Werbung hereinzuholen, aber es ist jetzt entschieden, daß LinuxFocus so bleibt wie bisher. Wir werden keine blinkenden Werbebanner zeigen. Es soll dein Magazin sein und es sollte auch eine unabhängige Zeitung sein. Nach fast drei Jahren ist LinuxFocus noch immer das einzige freie, mehrsprachige Magazin über Linux und wir planen, auch in Zukunft unser mehrsprachiges Konzept beizubehalten.

Während du das Magazin liest, werden dir ein paar Änderungen auffallen:

  • Seit Mitte Januar hat jeder neue Artikel im Inhaltsindex auf der linken Seite einen Eintrag "Kommentar zu diesem Artikel". Durch Klicken auf diesen Eintrag kannst du jetzt Kommentare abgeben oder die Kommentare anderer Leser lesen.

  • Ja, LinuxFocus hat ein neues Logo. Das neue Logo zeigt Tux, den Linuxpinguin, wie er auf dem LF-Delphin reitet. Es gibt auch einen kleinen Knopf, den du dazu benutzen kannst, einen Link auf LinuxFocus zu setzen. Hierfür mußt du den folgenden Code in deine Webseite einfügen: 
    
<a href="http://www.linuxfocus.org">

alt="LinuxFocus" width="143" height="45"></a>

  • LinuxFocus hat eine neue Rubrik "Kurze Artikel und Tips". Dies ist eine Rubrik, wo jeder nützliche Tips zu Linux direkt online beitragen kann. Es muß nichts schwieriges sein. Es kann so einfach und nützlich sein wie dies:
    " Was war die Datei, die ich als letztes in diesem Verzeichnis angeschaut (nicht editiert) habe?
    Lösung: Benutze den Befehl ls -ut | head "
    • Momentan verfügbare Tips lesen.
    • Selbst einen kleinen Beitrag einreichen.
