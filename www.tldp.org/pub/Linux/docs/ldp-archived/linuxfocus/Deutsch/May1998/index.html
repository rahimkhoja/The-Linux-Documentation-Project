Mai 1998: 3D Bilder modellieren unter Linux

In dieser Ausgabe von LinuxFocus geht es um 3D Modellier-Programme für Linux. Anders als OpenGL und POVRAY werden Modellier-Programme zum Entwurf von 3-dimensionalen Objekten mit komplexer Geometerie und Textur verwendet.

Die Frontseiten-grafik dieser Ausgabe wurde von Armando Sepulveda gestaltet.

Inhalt

    3D Modellier-Software im Fokus

  • MindsEye: 3D Modeling von Martin van Velsen
    Seit etwas 2 Jahren gibt es ein Projekt namens MindsEye. Das Ziel diese Projektes ist, ein Modeling/Rendering/Animation Paket für Linux/Unix zu schaffen, das equivalent zu großen Programmen wie 3D Studio Max oder LightWave ist. Bis jetzt läuft das Projekt gut, obwohl es am Anfang einige Startprobleme gab. Einige Aspekte von MindsEye sind einzigartig unter den Modeling-Programmen.

  • RenderMan von Carlos Calzada Grau
    Wer kennt Pixar? Wer hat nicht "Toy Story" oder "Das große Krabbeln" gesehen? Pixar hat RenderMan, einen Rendering Interface Standard, entwickelt. RenderMan erlaubt es Modellierern zu spezifizieren, was gerendert werden muß, ohne genau festzulegen, welcher Algorithmus benutzt werden soll. Lern mehr über dieses fantastische Programm.

    • Hardware & Software Review

  • Microsoft Windows NT Server 4.0 im Vergleich mit UNIX , von John Kirch
    Das Ziel diese Artikels ist es, EDV-Managern in Unternehmen zu helfen, sinnvolle Entscheidungen beim Kauf von Computerhardware und Software zu treffen. Die Informationen in diesem Artikel basieren auf den Erfahrungen von erfahrenen Mirosoft Information System Professionals.

    • Die Kernel Ecke

  • Real-Time Linux (Echtzeit Linux) von Ismael Ripoll
    Der Autor gibt eine gute Übersicht über die Welt der Echtzeit-Betriebssysteme. Dieser Artikel ist für interessierte Leser ohne spezielle Informatikkenntnisse geschrieben.

  • Creating a (Somewhat Useful) Version of Linux: Ein nützliches Mini-Linux auf Diskette selbst erstellen, von Mooneer Salem
    Dieser Artikel zeigt, wie man unter Linux seine eigenen root/boot Disketten erstellt, um diese mit den, für seinen Rechner, wichtigsten Programmen für Notfälle und Reparaturarbeiten auszustatten.

    • Software-entwicklung

  • The Postscript Language von Emre Demiralp
    Dies ist der erste Artikel in einer Serie von Artikeln über Postscript. Das Ziel diese Artikels ist es, einen Überblick über die Sprache zu geben, so daß man Spaß am Erstellen von eigenen Dokumenten in dieser interessanten Programmiersprache hat.

  • GLUT: Verwenden von Menüs von Miguel Angel Sepulveda
    Unsere GL Utility Toolkit Serie wird hier fortgesetzt und zeigt dieses Mal, wie man die Menü-Programmierschnittstelle benutzt. GLUT Menüs erlauben es dem Entwickler, klare, kompakte und portierbare User Interfaces für OpenGL Applikationen zu schreiben.

  • OpenGL: Die 3D Szene von Miguel Angel Sepulveda
    Erforsche die Grundlagen des 3D Rendering mit OpenGL. Der Autor beginnt mit einem Beispiel einer 3D Animation. In dieser Ausgabe werden 3D geometrische Transformationen und Matrix Projektion behandelt.

  • Einführung in SQL: Installation von PostgreSQL von Manuel Soriano
    Dieser Artikel ist der erste Artikel in einer dreiteiligen Serie über SQL. Diese Serie wurde speziell für Anfänger im Bereich Datenbankentwicklung geschrieben.

    • Die Grafikecke

  • POVRAY: Basic Notions von Antonio Castro
    In unserem zweiten Artikel zu POVRAY diskutieren wir einige grundlegende Notationen und fangen mit einfachen Elementen dieser Sprache an.

  • Texte mit PhotoShop schreiben von Juan Manuel Sepulveda
  • Texte mit Gimp schreiben von Phil Ross
    In der Grafikecke vergleichen wir zwei gute Bildbearbeitungs Programme: Photoshop und Gimp. Wir untersuchen, wie man Text-Banners mit beiden Programmen erstellen kann.

    • System Administration

  • Einführung in DNS von Andreas J Gundacker
    Dieser Artikel ist für all jene geschrieben, die sich für das Gebiet der weltweiten Computernetze interessieren. Du hast Dich vielleicht schon gefragt, was eigentlich passiert, wenn man in Netscape einen URL eingibt. Dieser Artikel wird Dir helfen zu verstehen, wie der name-lookup Prozeß funktioniert.

  • Der X Display Manager von Joel McCarty
    Der X Display Manager läuft als ein Hintergrundprozeß und managed mehrere X-Display Verbindungen zu lokalen oder zu entfernten Rechnern. Er stellt einfaches Session Management zur Verfügung.

    • Anwendungen unter Linux

  • StarOffice für Linux von Ismael Ripoll
    Dieser Artikel befaßt sich mit den Applicationen des StarOffice Paketes. Der Autor testet, wie man eine Präsentation mit StarOffice erstellen kann.
