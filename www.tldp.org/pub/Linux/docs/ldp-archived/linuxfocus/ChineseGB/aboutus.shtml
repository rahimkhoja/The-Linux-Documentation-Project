关於 LinuxFocus

LinuxFocus 创刊於 1997 年十一月。一开始 Miguel Angel Sepulveda 和一些朋友感觉到非英语的 Linux 文件实在太少了，因此他们决定成立这本多国语言的杂志。

虽然我们大多数人都会读英文文件，书籍和新闻组.但我们觉得， 以非英文的方式来提供资讯十分的重要。 许多在非英语系国家的年轻人所学的英文有限， 他们第一次接触电脑的经验很可能就是微软的操作系统。 微软的操作系统已经被翻译成几乎世界上任何一种语言的版本，而商业化的 Linux 在这方面还相差甚远。

因此我们的理想是希望能将这种方式发扬光大，使更多人能用他们自己语言来取得信息， 进而建立新一代的 Linux 使用者。

Miguel Angel Sepulveda 是 LinuxFocus 杂志的创始人。他原来在 Pittsburgh 大学做聚合体和量子的研究工作。 他有一个理论化学的博士学位，且钟爱动画,3D 立体程式设计和模拟。 自从 1993 年成为一个 Linux 的使用者以来， 就很少有时间再听他喜爱的古典音乐和带小孩。 他於 1999 年辞去了 LinuxFocus 领导的职务。


LinuxFocus 的目标是出版一份免费的网上 Linux 杂志。LinuxFocus 是一份公开的、 多语的杂志，它是由许多人共同努力的成果，任何人也都可以投稿自己的文章。

Guido Socher 是继 Miguel Sepulveda 离开之後 LinuxFocus 的领导人。

Guido Socher Guido 从他第一次安装 Linux 在他的电脑上就成了一个标准的 Linux 迷。 他喜欢 Linux 不只是它的软件，而是它的哲学。Guido 是一位电子工程的硕士， 目前在一家电信公司担任软件设计的工作。最近他将很多的空馀时间都花在 LinuxFocus 上。




LinuxFocus 绝对不是一个人的成果，而是集数百人的贡献而成的。 我们还有数组负责不同语言的翻译人手。
每一组的联络人可以在我们的 lf-team 网页 上找到。

我们竭诚希望您能够享受阅读这份杂志，更欢迎您的建议和批评指教。
