LF Tip: extensiones de Firefox imprescindibles

Resumen:

Introducción

Mozilla Firefox está definitivamente preparado para "la Red". Algunas páginas que ofrecen información útil tienen sin embargo alguno molesto y publicidad no deseada. A continuación dos extensiones que uso cuando navego "la salvaje Red".
 

Flashblock

Flashbloc quita todos los plugins de flash y los reemplaza por un pequeño botón redondo. ¡No mas imágenes de flash parpadeantes!. Si quieres ver la animacionón de flash entonces pulsa con el ratón en el pequeño botón y comenzará.

https://addons.update.mozilla.org/extensions/moreinfo.php?id=433


 

NukeAnything

Nuke Anything puede quitar cualquier objeto html. Todo lo que tienes que hacer es pulsar con el botón derecho del ratón sobre el objeto, el cual puede ser incluso un párrafo de texto y seleccionar "remove object". Muy útil para quitar imágenes y cualquier tipo de publicidad. También se puede usar para limpiar una página antes de imprimirla.

https://addons.update.mozilla.org/extensions/moreinfo.php?id=79
