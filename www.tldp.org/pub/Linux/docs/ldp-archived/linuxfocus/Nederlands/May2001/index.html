LinuxFocus Mei 2001

Je grootmoe wilt niet overstappen op Linux?

Er zijn, net als haar, nog veel Windows-gebruikers die aarzelen om Linux te installeren. Misschien hebben ze Windows al 5 keer opnieuw geinstalleerd en zijn ze het nu gewoon beu en berusten ze erin dat hun installatie nu toch meestal werkt. Dit is dikwijls het geval, maar de meesten onder hen zijn eigenlijk toch wel nieuwsgierig naar wat Linux te bieden heeft.

SuSE biedt hiervoor een oplossing aan onder de naam SuSE live evaluation CD. Dat is een complete Linux distributie, voorgeïnstalleerd op één enkele CD-ROM. Je kunt booten vanaf de CD maar als dat niet lukt kun je ook een boot-floppy maken.

Je hebt Live CD's vanaf SuSE 7.0; de meest recente versie is "live-eval-7.1" met, natuurlijk, SuSE 7.1.

Eerste krijg je een gewoon Yast2 installatie-scherm. Toch is het geen echte installatie: je moet enkel de configuratie doen. De installatie maakt wel 3 bestandjes aan op de eerste partitie van je IDE schrijf. Het maakt daarbij niet uit of deze partitie ext2 of het Windows bestandssysteem bevat; het werkt allemaal. Dit zijn de 3 bestanden:

  • suselive.710, ongeveer 2 MB - bevat data voor de SuSE Linux 7.1 Evaluation CD.
  • suselive.swp, ongeveer 80 MB - een swap file.
  • suselive.usr, ongeveer 100 MB - een ext2 bestandssysteem-in-een-bestand met de /home directory van de gebruiker

Met deze Live CD kun je dus zonder risico een volledig Linux systeem uitproberen. Geen gepartitioneer meer, geen verplaatsen van data. Enkel die drie bestanden en als je ze verwijdert blijft er geen spoor meer achter.

Ook als je zelf al Linux draait is dit een handige manier om een nieuwe distributie te proberen. Probeer eerst, en beslis daarna of je wilt overstappen of upgraden. En de nadelen? Wel, het draait vanaf de CD en daardoor is het dus wel trager dan een normale installatie op harde schijf.

Je kunt een ISO image afhalen van ftp://ftp.suse.com/pub/suse/i386/live-eval-7.1/ . Of probeer een mirror dichter bij huis, want SuSE's site heeft het soms moeilijk. Als je een kabel/DSL connectie moet missen, kun je bij SuSE een cd'tje bestellen of misschien kan een vriend je wel een schijfje schrijven. Misschien vind je wel een magazine met een gratis CD bij?

Dit is duidelijk een uitstekende manier om Linux te promoten onder vrienden, zodat niet enkel je grootmoeder Linux-fan wordt en overstapt ;-)

De LinuxFocus Tip

Hoe print je verschillende pagina's op 1 enkel blad? Probeer dit. Print vanuit Netscape (of een ander programma) en vervang het print commando door:
psnup -2 | lpr
Dit voorbeeld drukt twee pagina's af op 1 blad. psnup maakt deel uit van een hele set printopdrachten en postscript toverijen genaamd 
psutils
(te krijgen op http://www.dcs.ed.ac.uk/home/ajcd/psutils/). Een ander leuk voorbeeld is psbook. Het herschikt de pagina's zo dat je een A5 boekje krijgt als je 2 pagina's per blad afdrukt. Je moet enkel nog het papier middendoor vouwen en nieten. Je hebt wel een duplex printer nodig (die het papier op beide zijden bedrukt.) Het commando is:
psbook | psnup -2 | lpr -Pduplex In dit voorbeeld heet onze duplex printer duplex. Maar ook zonder duplex printer is het handig! Deze opdracht converteert A4 pagina's naar het US-letter formaat:
psresize -PA4 -pletter in.ps out.ps
