Schrijf je in op LinuxFocus announcements

Lezers die op de hoogte willen blijven kunnen zich inschrijven op de "announce" mailing list.

Je mail-adres zal aan niemand getoond worden (behalve LinuxFocus medewerkers dan.) Je kan op elk moment je inschrijving ongedaan maken.

Als je enkel een tijdelijk mail adres hebt (bv. op je universiteit) denk er dan aub aan om uit te schrijven voordat het adres ongeldig wordt.

Vraag: Hoe kan ik inschrijven?
Antwoord: Stuur een e-mail naar majordomo@linuxfocus.org met als bericht (niet als titel, die is niet belangrijk):

subscribe announce 
end
Vraag: Hoe kan ik uitschrijven?
Antwoord: Stuur een mail naar majordomo@linuxfocus.org met als bericht (ook hier weer niet als titel):

unsubscribe announce je_mailadres@weetikwaar.com
end
