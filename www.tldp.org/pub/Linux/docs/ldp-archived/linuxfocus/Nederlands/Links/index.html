LinuxFocus Links

LinuxFocus heeft niet de bedoeling om links te geven naar alle Linux-sites. Daar zijn andere sites voor. Toch willen we voor de geïnteresseerde lezers een klein overzicht geven van interessante startplaatsen op het wereldwijde Linux web.

Als er een link niet werkt, of als je een link wil toevoegen, contacteer ons dan op dutch@linuxfocus.org
We zoeken vooral links naar gratis Linux magazines en naar niet-Engelstalige Linux sites.
Bedankt!
