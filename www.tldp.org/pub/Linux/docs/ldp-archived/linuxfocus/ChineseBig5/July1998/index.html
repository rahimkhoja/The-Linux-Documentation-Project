LinuxFocus July 1998

新的一期出版囉... 這是第一份多國語文版本的 Linux 免費線上雜誌， 以 Linux 使用者為對象， 並由 Linux 愛好者所撰寫。 在七月號的封面裡， 我們的 LF 吉祥物正在前往加洲海灣區的路上， 代表著我們的雜誌完成一個章節， 正準備開啟歷史性的新頁。 最近 LinuxFocus 的網頁主機， 正準備由匹茲堡移往海灣區， 這是一項令人振奮的消息， 並將會大大增加矽谷內， Linux 使用者的數目與力量。 LF 會持續盡最大的努力， 為讀者帶來深入的報導， 以及各式來自 Linux 愛好者的消息。
感謝 UPV
本期新的雜誌裡， 我們帶來了一些有趣的文章， 內容涵蓋影像處理到系統安全。 在此感謝所有作者， 與我們分享他們的知識與作品， 但在這個月當中， 我們要特別感謝來自 Polytechnic University of Valencia (UPV) 裡， 資訊科學系的師生們， 他們提供許多文章， 以及協助翻譯工作等。
最新之 Linux 產品與服務

CrossWind Technologies 最近發表了 (June 23 , 1998) Linux 版本之 CyberScheduler 2.1， 這是一個網頁式的群組工具軟體， 具有月曆及行事曆排程功能， 可在 Apache 網站伺服器上執行， 並支援各式瀏覽器程式。 目前可採用 RPM 或 TAR 檔案方式來安裝。

又有個新的 Linux 聊天室上網了， 提供有關 Linux 話題、 疑問、 線上文件等的討論。 它的名字就叫做 Linux-Chat， 您可以透過 telnet (linux.intergrafix.net 5000) 或 java applet 方式， 來連上聊天室。

諮詢網頁 (Consultants Page) 新構想! 由於 LF 雜誌並未向讀者收取任何費用， 所以我們也沒有經費， 來支付報酬給作者及翻譯者。 但是， 對於許多位貢獻無比心力， 促成雜誌不斷進步的無名英雄， 我們仍舊深感虧欠。 從 1998 九月開始， LF 希望能夠開放一個「諮詢網頁」， 每位雜誌的作者與計畫協助者， 都可以擁有一個小空間， 放置他們的廣告或介紹， 像是專長、 電子郵件信箱、 網頁位置等。 我們希望這項新服務， 不但能成為計畫協助者的良好回饋方式， 所累積的資訊， 也能成為 Linux 世界裡的重要資產。

什麼是 LinuxFocus? 我們是一個由全球 Linux 愛好者， 所組成的團體， 鑑於 Linux 使用群日益龐大， 我們成立的目標， 就是要做出最好的 Linux 線上雜誌。 雜誌內的所有文章， 都是以多國語文寫成， 藉以協助引領新手進入 Linux 與 UNIX 的神奇世界。 有別於網路上其他現有的雜誌， 我們並非定位在「秘技絕招」的收集上， 正如雜誌的名稱所示， 我們提供「深入主題」的資訊， 針對特定的主題， 以系列性的文章加以介紹， 如果使用者隨著主題， 系列式地閱讀， 必能由新手轉變成為進階使用者。

LinuxFocus 支援多少國家的語文? 這點很難確定， 因為隨著口耳相傳的結果， 我們的成員數量正持續擴大。 目前已有四個堅強而穩定發展的翻譯團隊： 英國、 法國、 德國、 西班牙， 另外還有幾個規模較小、 人數還在增加的團隊： 俄國、 義大利、 萄葡牙、 荷蘭， 期待這些團隊能日益壯大及穩定。 對於新的朋友， 我們永遠敞開大門歡迎， 如果您有意成為 LF 的一份子， 請查看「關於 LF」一文， 與適當的主編連絡， 並寫信給我們， 竭誠歡迎新朋友的加入。

關於版權與許可 我們網站上的所有內容， 其版權都屬於每位作者個別所有， LinuxFocus 並未擁有這些文章的版權。 由 LF 所提供之雜誌內容， 是在作者的默許下， 進行公開發行的， 同時， 作者也授權同意翻譯， 在文章未被擅改之下， 隨著本雜誌的相關內容， 可以透過全球資訊網、 檔案傳輸服務、 或是其他傳播媒介， 自由地發行。 LF 僅允許雜誌內的文章被用於非營利場合， 如果想要在商業場合， 引用 LF 的文章或譯作， 那麼您應該明確地向原作者， 或是向 LF 的編輯群徵詢同意。
