LinuxFocus: November 1997

Contents

  • What is Linux? by Miguel Torrealba
    A brief history of Linux
  • Installation of Manpages-es by Miguel Angel Sepulveda
    The author describes the installation and configuration of the spanish online manual pages
  • Conneting to the Internet by Manuel Trujillo Albarral
    A step by step procedure to connect your linux box to the internet using a ppp protocol
  • I got Linux Installed, What next? by Jose Maria Laveda
    Some information on resources available to new users
  • Some Tips by Manuel Trujillo Albarral
    Some tips and answers to common problems
  • Making Shared Libraries by Luis Colorado
    All what you wanted to know about shared libraries
  • PGP Under Linux by Angel Lopez
    The author takes us on a tour of PGP
  • Managing Your Mail with procmail by Angel Lopez
    Sort your mail into folders, automatic replies, mailing lists, learn how to do these and more using procmail
  • Linux in Hospitals by Manuel Soriano
    Linux means business, see how linux is been used in a healthcare environment
