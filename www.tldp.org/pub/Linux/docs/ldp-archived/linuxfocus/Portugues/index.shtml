O que é a LinuxFocus?

A LinuxFocus é uma revista internacional livre para o Linux. É uma organização sem lucros e a revista é administrada por voluntários a partir de todo o mundo. A LinuxFocus fornece documentação livre para um sistema operativo livre. A LinuxFocus é internacional e os nossos artigos são traduzidos em línguas diferentes.
Divirta-se a ler os artigos da LinuxFocus!
