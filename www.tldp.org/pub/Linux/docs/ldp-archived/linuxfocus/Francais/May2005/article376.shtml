Introduction

Mozilla Firefox est totalement prêt pour le « world wide web ». Certains sites pourtant très intéressants se voient cependant parfois pollués par des publicités agressives et dérangeantes. Voici donc deux extensions que j'utilise lorsque que je surfe sur le "world wild web".
 

Flashblock

Flashblock retire tous les greffons flash et les remplace par un petit bouton rond. Plus de images flash qui clignotent ! Si vous souhaitez néanmoins voir l'animation flash, cliquez simplement sur le petit bouton et elle démarrera.

https://addons.update.mozilla.org/extensions/moreinfo.php?id=433


 

NukeAnything

Nuke Anything peut retirer n'importe quel objet html. Tout ce que vous avez à faire est de faire un clic droit sur l'objet, qui peut même être un paragraphe de texte, et sélectionnez « Remove this object ». Très utile pour retirer les images et les publicités. Il peut également servir à « nettoyer » une page avant de l'imprimer.

https://addons.update.mozilla.org/extensions/moreinfo.php?id=79
