1. New documents

1.1. Fresh@LDP

Two new documents this week:

  • Review Howard Shane's Motorola Surfboard 4100 + 4200 Series USB Cable Modem mini-HOWTO if you want to know how to set up this type of cable modem.
  • By the same author, we published the Scanner HOWTO, a long-time black hole in the Linux documentation which is now being filled.

1.2. Proposed new documents

  • Same goes for Daniel Quintiliani, who proposed a Xnews-under-Linux-HOWTO.
  • Rick Meyerhoff proposed to write an InstallFest-HOWTO.
  • Timothy M. Lyons wanted to contribute a new mini-HOWTO on using grub to multi-boot into RH9/Win2K/WinXP and using VMware and Raw disks to utilize any alternate OS's while in any one of the others.

1.3. Submitted for review

  • Salvador Peralta posted a Scripting Graphical commands with Tcl/Tk mini-HOWTO, but Tabatha asked to send it to the general discuss mailing-list first for discussion.
  • Saqib Shaikh has been maintaining a document called the "BLinux Software Map" which aims to give an objective overview of software currently available to help blind users use Linux and would like to submit to the LDP after review. If you are interested in reviewing this document, get in touch with <tabatha@merlinmonroe.com>.
  • core_dump posted his Xnews-under-Linux-HOWTO for review. The document describes how to get the Windows newsreader Xnews to work under Linux and deal with problems that may arise.
  • David Braun's Disk Encryption HOWTO has been converted to DocBook. And David has requested for feedback.

2. Updates

2.1. HOWTOs

  • Another update for Miroslav Skovic's FBB Packet-radio BBS mini-HOWTO, which is now v1.23. By the same author v 3.12 of the LILO mini-HOWTO and the Linux+WindowsNT mini-HOWTO v2.12.
  • By Saqib Ali: the Apache based WebDAV Server with LDAP and SSL, v4.0.2.
  • From Stephen A. Zarkos, we received the Sentry Firewall CD HOWTO, v1.3.1.
  • The Linux Ethernet-HOWTO by Paul Gortmaker, v2.9 is now available.

2.2. Guides

  • M. Leo Cooper sent in v2.0 of the Advanced Bash Scripting Guide.

2.3. Various

  • Paul Sutton updated his JOE Command Reference to v3.0.
  • It appears that the heat has done our Spanish colleagues much good ;-) So much indeed, you better take a look yourself this time at the new items listed at http://es.tldp.org.

3. Happenings in the LDP world

3.1. LDP Author Guide

After about 30 hours of hard work, Emma Jane Hogbin commited the changes that she had planned for the LDP-Author-Guide. You are allowed to poke around but Emma would prefer that no major changes apart from the re-organization that she had earlier suggested be done until the proposed task is complete.

4. Feedback and Contributions

You can help The Linux Documentation Project by:

  • Reviewing one of the draft versions of the documents in progress.
  • Picking up one of the unmaintained documents.
  • Providing assistance with markup.

Please help us create this weekly newsletter. Seen a site putting in a story with a link to the LDP? Completed the translation of any HOWTO? Know of any articles about the LDP published in newspapers or magazines? You can update us by sending in links and information at <feedback@en.tldp.org>.
