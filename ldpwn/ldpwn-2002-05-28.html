LDP Weekly News
2002-05-28

by David Merrill
 

New Documents

  • ADSL Bandwidth Management HOWTO
    This document describes how to configure a linux router to more effectively manage outbound traffic on an ADSL modem. Emphasis is placed on lowering the latency for interactive traffic even when the upstream bandwidth is fully saturated.
    http://tldp.org/HOWTO/ADSL-Bandwidth-Management-HOWTO/
    Version 0.1, Dan Singletary (Author) dvsing@sonicspike.net

  • KDE GUI Login Configuration HOWTO
    This is the KDE GUI Login Configuration HOWTO, a tutorial on customiz? ing the GUI login screen. Topics include: How to add other window managers to the drop-down selection list; how to enable user selection icons in the login window; and requiring root permission for system shutdown.
    http://tldp.org/HOWTO/KDE-GUI-Login-Configuration-HOWTO.html
    Version 0.03, John P. Meshkoff, II (Author) johnm@sivakalpa.org

  • Querying libiptc HOWTO
    This HOWTO explains how to use the libiptc library included in the iptables package. This document can show you how to use short C or C++ programs to query the internal structure of the firewalling code, to check chains and rules, packet and byte counters, and in a second phase, if you are a little "brave", to modify them.
    http://tldp.org/HOWTO/Querying-libiptc-HOWTO/
    Version 0.1, Leonardo Balliache (Author) leonardo@opalsoft.net

  • Remote Bridging with IP Tunnels mini-HOWTO
    This document describes how to unite two separate ethernet LANs with an IP tunnel between them.
    http://tldp.org/HOWTO/mini/Remote-Bridging/
    Version 1.0, Alexander Alekseev (Author) alex@zapad.msk.ru

  • VideoLAN Howto
    This howto describes how to use the complete VideoLAN solution.
    http://tldp.org/HOWTO/VideoLAN-HOWTO.html
    Version 1.1.2, Alexis de Lattre (Author) alexis@videolan.org, Marc Ariberti (Author) marcari@videolan.org

  • VideoLAN Quickstart
    This document describes how to start using VideoLAN quickly.
    http://tldp.org/REF/VideoLAN-Quickstart/
    Version 0.8, Alexis de Lattre (Author) alexis@videolan.org, Bill Endridge (Author) bill@rfa.org

About the LDP

The Linux Documentation Project is developing free, high quality documentation for the GNU/Linux operating system. This includes the creation of HOWTOs and Guides, and collaboration with other documentation groups.

If you've always wanted to help Linux reach Total World Domination(tm), but you're not a programmer, there's still something you can do. Help the LDP!

The LDP keeps a page of resources for authors at http://www.tldp.org/authors/. Contributions are always welcome.

For more LDP Weekly News, go to http://www.tldp.org/ldpwn/
