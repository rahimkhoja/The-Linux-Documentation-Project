LDP Weekly News
2001-02-27

About the LDP

The Linux Documentation Project is developing free, high quality documentation for the GNU/Linux operating system. This includes the creation of HOWTOs and Guides, and collaboration with other documentation groups.

If you've always wanted to help Linux reach Total World Domination(tm), but you're not a programmer, there's still something you can do. Help the LDP!

The LDP keeps a page of resources for authors at http://www.linuxdoc.org/authors/. Contributions are always welcome.

For more LDP Weekly News, go to http://www.linuxdoc.org/ldpwn/

Document Review Effort Update

The documentation review effort is proceeding apace. Our focus at the moment is on establishing measurement criteria and working out the details of the process. Also, we are updating all documents to the author's latest version, where necessary.

We are also developing a database to store document metadata and reviewing notes. This database will be a tool for all LDP reviewers to coordinate and record results. Rick Welykochy is developing a web front end in perl. It is in a very early alpha right now, but we hope to have it functional within a few weeks.

The mailing list for this effort is currently at http://www.lupercalia.net/mailman/listinfo/ldp-review, and we very much welcome new volunteers. Interest has been very encouraging so far.

Please write david -AT- lupercalia.net for more information about how you can help.
