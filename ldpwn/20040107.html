1. New document proposals

  • Nathan Rosenquist submitted a draft for a HOWTO accompanying the rsnapshot program. He awaits your comments.

2. Updated HOWTOs, FAQs and Guides

  • The Windows Newsreaders under Linux HOWTO v2.11 by Daniel Quintiliani focuses on getting the sound right.
  • From Rick Moen we published v1.7.4 of the Linux User Group HOWTO and v1.4.16 of the WordPerfect on Linux FAQ.
  • Greg Lindahl's MS Outlook to Unix Mailbox Conversion mini HOWTO v1.3 details the conversion of MS Outlook (not Outlook Express) mailboxes to UNIX mailboxes using Mozilla or IMAP.
  • More nifty things you can do with Bash in Mendel Cooper's Advanced Bash-Scripting Guide v2.3.
  • The Linux Loadable Kernel Module HOWTO by Bryan Henderson is now at v1.05. This revision has information on module.h and -DMODULE, typo fixes and adds information on kallsyms.
  • The revised LDP Author Guide with the separated sections on authoring process (main guide) and technical HOWTOs (appendix). Now at v4.0.11.
  • Machtelt Garrels's Introduction to Linux - A Hands on Guide has been updated with a few changes in the document for better presentation of the content. The document is now at v1.9

4.1. LDP unreachable author problems

Jean-PhilippeGuérard, the howto translations coordinator for the French project traduc.org said that traduc.org would like to publish the Compressed-TCP Sessions using SSH-like tools HOWTO but wanted information regarding the license under which it was published and was not able to reach the author. David Lawyer responded saying that The LDP was facing similar problems and that they would be of little help.

5. HOWTO contribute to TLDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
