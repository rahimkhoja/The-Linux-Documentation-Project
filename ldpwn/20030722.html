2. Updates

2.1. HOWTOs

  • If you want to do strange things with strange video hardware, such as working with very high or very low resolutions which are not supported by default in the standard XFree86 package, Eric Steven Raymond's XFree86 Video Timings HOWTO might be helpful. However, check with Chapter 2 first to see whether there's not a more elegant alternative available.
  • Saqib Ali's DocBook XML/SGML Processing Using OpenJade documentation v2.2.8 now includes additional information for Cocoon, the Apache XML publishing framework, which can be integrated with Tomcat, the Java servlet container, to set up your state-of-the-art application server. The important thing is to keep your JRE up to date!
  • Stephen Darlington's Oracle 8i for Linux Installation HOWTO is now v1.18.
  • The FBB Packet-radio BBS mini-HOWTO by Miroslav "Misko" Skoric is at v1.21
  • Machtelt Garrels updated the Introduction to Linux - A Hands on Guide to v1.6 with a few corrections in the initial chapters.

Charles Curley installed OpenOffice.org 1.1RC and wrote a very simple test document in OOo's native format and posted his notes regarding the same. While on thread, Chris Karakas said that he would document the procedure of writing the entire document in LyX, exporting it to SGML and post-processing to convert it to various formats.

3.3. Linux One Stanza Tips (LOST)

The LOST (Linux One Stanza Tips) project is a series of small Linux related hints and tips and a set of of scripts/programs for interfacing the snippets with existing mailers and web pages. The LOST project was born in the Linux-India-Help mailing list in Aug 2001 because the developers of the project felt that as newbies join the list, the same queries are floated time and again.
