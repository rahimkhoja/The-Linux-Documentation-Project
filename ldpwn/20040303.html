1. New documents @ TLDP

  • As proposed a couple of weeks ago, here it is: the DVD Playback HOWTO, by David Jao. David starts with the basics and explains how to set up the region for your drive, how to check for X Video Overlay support and how to enable DMA drive access.

    He continues with the discussion of the DVD playing software: MPlayer, Xine, Ogle and VideoLAN. The HOWTO has information of how to make these work on a variety of Linux distributions. It also has a troubleshooting section.

  • At the Spanish TLDP you can now find two courses about the Andalusian Guadalinex distribution. The first document is a guide detailing the installation process and explaining how to get started with the system. It also has an introduction on Internet services and OpenOffice. The second document is a beginner's guide as well, but it differs from the first one in that it has more on multi-media and didactical applications.

2. Documents submitted for review

  • AJ Lewis sent in an update of the LVM HOWTO. This revision has updated CVS locations and FTP links along with an added section on extending a JFS filesystem

3. New document proposals

  • Ravi Kiran asks whether this article on understanding the proc file system would be a good addition to our collection. Ravi would also like to hear your suggestions on this tutorial on writing a file system.

4. Updated HOWTOs, FAQs and Guides

  • Eric Raymond's UPS HOWTO v1.1 has a whole new section entitled "An Overview of Power Protection", explaining the differences between surge suppressors, line conditioners and un-interruptible power supplies. Furthermore, Eric added a section on the deployment of your UPS.
  • Also by ESR: v3.0 of the Unix Hardware Buyer HOWTO. It has been adapted to the 21st century and the Ultimate Linux Box articles have been integrated in this HOWTO.
  • If you were wondering how to use your Linux box for controlling your telescope, check out the Astronomy HOWTO (v1.48) by Elwood Downey and John Huggins.
  • The Time Precision HOWTO v1.0.6 by Avi Alkalay has been translated into Russian, thanks to Shaki.
  • Later on this week, we received another update from Eric Raymond, this one for the DocBook Demystification HOWTO. V1.3 features updated links, corrected figures and additional pointers to XML editors.
  • Eric was in a very productive mood: we also received v1.6 of the Linksys Blue Box Router HOWTO. As you recall, this document describes the BEFSR41 models and up of Linksys router/firewall boxes, which are very suitable for use with a home DSL connection. It also explains firmware upgrades.
  • The K7s5a HOWTO v1.8, a combined effort by Sebfrance, Andreas Willenbrink, Alvaro Reguly and Paolo Berva has a new section on making the sensors of this motherboard work with the 2.6 kernel.

6. HOWTO contribute to TLDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
