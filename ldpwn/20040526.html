1. New document proposals

  • Artemio has been working with RTLinux for a while, and wrote installation instructions to help newbies on the RTLinux mailing list. His RTLinux installation instructions are now also a part of the official RTLinux package. Artemio would now like to take over the (out dated) RTLinux HOWTO and create a RTLinux Installation HOWTO and RTLinux Programming HOWTO (or, RTLinux Module Programming HOWTO).
  • Teus Benschop proposed to write a Bible Translation HOWTO. His HOWTO would cover the software that is currently available for Linux, explain how VMware was used to use to run the Bible translation software that is available for Windows, and detail out how to integrate the two OSes to make the best of the job, and then call for new software to be written.

2. Updated HOWTOs, FAQs and Guides

  • The K7s5a HOWTO describing the Elite K7s5a motherboard is a joint effort of Sebfrance, Andreas Willenbrink, Àlvaro Reguly and Paolo Berva. Version 2.0 contains corrections sent in by the readers.
  • Ricardo Alexandre Mattar posted v1.2 of the Samba Authenticated Gateway HOWTO. This HOWTO documents the creation of a firewall or gateway that uses rules based on user access, in which the users are authenticated by a Samba Primary Domain Controller. On the outside, you don't notice too many changes; this version is mainly a cleanup of the source code.
  • Machtelt Garrels sent in v1.13 of the Introduction to Linux guide. Apart from some minor updates, she added references to rdesktop, updated the glossary and thanks the people who translated the guide into Hindi.
  • The Linux Documentation Project Reviewer HOWTO v1.4.2 corrects some spelling mistakes from v1.4.1. Emma Jane Hogbin sent in this new revision.
  • Artemiy I. Pavlov released the SquashFS HOWTO v1.1. This revision has modifications to the HOWTO to keep it current to SquashFS release 1.3r3.

3. News in The LDP world

  • New mirror in Belgrado.

    Ljubisa Radivojevic volunteered to set up a mirror of TLDP in Belgrado. With this, he created the first mirror in the Serbia-Montenegro region. A complete list of our mirrors can be found at http://tldp.org/mirrors.html. Most are updated daily or even multiple times per day, so choose one near to you!

4. Discussions on The LDP lists

  • Bible Translation HOWTO

    Teus Benschop proposed to write a Bible Translation HOWTO and requested feedback. People had different ideas about the topic and the content of documents that The LDP should host. Rodolfo J Paiz posted his ideas on why Teus Benschop's proposal should be given a chance. Greg Ferguson said that it would be good if Teus could design this in a generic manner covering available translation software and then, cover the bible translation as a "case study".

5. HOWTO contribute to The LDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
