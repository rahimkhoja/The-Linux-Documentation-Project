1. New documents @ The LDP

Two new documents were added to our collection this week:

  • Eric Meyerhoff published v1.0 of the Linux Installfest HOWTO, which provides guidelines for running a successful installation party.
  • Alex Brennen polished and completed his Kerberos Infrastructure HOWTO so that it can serve the general public. We have v2.0 of this document for you, describing design and configuration of a Kerberos infrastructure. The document starts out with an introduction to Kerberos and a theoretical overview of the mechanism, and then details server setup and client configuration.

2. Documents submitted for review

  • There has been some discussion lately over what should be in the LDP FAQ. Rahul Sundaram summarized all the remarks and submitted them in the form of an update to the FAQ.

3. Updated HOWTOs, FAQs and Guides

  • Emma Jane Hogbin submitted v1.5 of the ACPI HOWTO The new version has minor updates for the 2.6.6 kernel and corrections regarding which kernels need patching.

4. News in The LDP world

  • Two new mirrors

    Gerardo Di Giacomo (aka Astharot) volunteered to set up a mirror in Tallinn, Estonia. This is our second Estonian mirror.

    Markus Markert is currently setting up a new mirror in Düsseldorf, which will soon be available here.

  • News from Italy

    Emilio Conti translated the Advanced Bash Scripting Guide, which is called Guida avanzata di scripting Bash in Italian.

    The PLUTO translated HOWTOs section contains quite some recent entries as well.

5. HOWTO contribute to The LDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
