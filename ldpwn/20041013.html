1. New documents @ The LDP

This week brings us, once again, a new addition to our collection:

  • Venkata Ramana Enaganti finalized the Autodir HOWTO, which explains autofs and how it can be used to create directories in a transparent manner. It also details the disadvantage of using the pam_mkhomedir module, as well as where it is different from autofs.

    According to the author: "Autodir offers a simple and effective means to create directories like home directories in a transparent manner. It relies on the autofs protocol for its operation. This document explains how to create directories on demand using Autodir in a transparent way to the applications. This document also explains using transparent backup feature that is possible with Autodir without bringing system down for backup purpose for all directories managed by Autodir."

2. Updated HOWTOs, FAQs and Guides

  • Miroslav Skoric updated his trio, to note the FBB Packet-radio BBS mini-HOWTO, the LILO mini-HOWTO and the Linux+WindowsNT mini-HOWTO.
  • Ryan Cartwright sent in an updated for the Windows LAN server HOW-TO, which fixes outdated references.
  • WordPerfect for Linux continues to be popular, and is still conditionally available. Consequently, Rick Moen keeps his WordPerfect on Linux FAQ up to date.
  • AJ Lewis has updates on LVM2 and on how to move a disk-group to another system in his http://tldp.org/HOWTO/LVM-HOWTO/.
  • Christoph Baumann contributed his Linux Touch Screen HOWTO under the GNU Free Documentation License.

3. News in The LDP world

  • Problems with WWW-HOWTO

    Doug Jensen reported the following problems with the WWW-HOWTO in the current LDP collection:

    • No attribution to the author, Pramod Karnad.
    • Removal of the Intranet-Server-HOWTO licensing.
    • Coping of Intranet-Server-HOWTO without proper attribution.
    • Introduction of a new license (an old version of LDP license).
    • Multiple licenses, without clearly stating how they are applied.
    • Lack of proper maintenance.
    Doug indicated that he would like the HOWTO to be removed and possibly reverted to a version similar to http://hrast.pef.uni-lj.si/docs/en/www_howto/WWW-HOWTO.html.
  • Linux HOWTOs in Swedish

    The Linux HOWTOs in Swedish package (doc-linux-sv) in debian has been orphaned. Anybody who is interested in maintaining the package should take a look at http://www.debian.org/devel/wnpp/index.html#howto on howto adopt a package properly.

4. HOWTO contribute to The LDP

The Linux Documentation Project (TLDP) is working on developing free, high quality documentation for the GNU/Linux operating system. If you want to help TLDP, consider joining <discuss-subscribe@en.tldp.org>.

Help us create this newsletter. We need volunteers who follow the community (mailing lists, newsgroups, web channels) and report events related to free software documentation. Write to us at <feedback@en.tldp.org>.
