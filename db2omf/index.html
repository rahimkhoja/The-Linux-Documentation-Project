NAME

db2omf - utility to extract OMF meta-data from DocBook documents.

SYNOPSIS

db2omf [OPTION] [FILE]

DESCRIPTION

db2omf extracts OMF meta-data from a DocBook document and writes it to a file in SGML format.

OPTIONS

-o filename write to the specified file.

BUGS

None currently known. Please report bugs to the author at:

<gferg@metalab.unc.edu>.

NOTES

db2omf was developed as a project of the Linux Documentation Project. It was originally written to create OMF files for submission to the ScrollKeeper system, the standard documentation meta-data database in many Linux distributions.

SEE ALSO

See the home page of the Linux Documentation Project, http://www.tldp.org for updates and more information. Also see the home page for OMF, http://www.ibiblio.org/osrt/omf/.
