Business and Commercial Resources:

  • ITtoolbox Portal, Content, community, and service for Linux professionals. Providing technical discussion, job postings, an integrated directory, news, and much more.
  • Linux in Business, such as mission-critical applications, industry and government.
  • Linux Enterprise Computing is a great page of resources for Linux in business, commercial, and enterprise settings.
  • Linux Ports, a site for Linux professionals and companies.
  • Red Hat support has lots of information.


Documentation and Tips:

  • About Debian Linux Guides, Step-by-step guides on installation and setting up LAN, Internet, and proxy servers and firewalls with tutorials on networking and compiling software.
  • "DocBook: The Definitive Guide".
  • The Linux man pages in a searchable (HTML) format.
  • The Linux Newbie Administrator Guide.
  • Linux Tips & Tricks, has tips and little hacks in a variety of categories.
  • Linux-Oriented Web Page, has a few links and some Linux documents.
  • The One Page Linux Manual, a summary of useful Linux commands (PS and PDF versions are available).
  • Red Hat Documentation contains a vast amount of help.
  • SGI TechPubs Library, contains Linux man pages, howtos, FAQs and various manuals (including the LDP Guides) in a searchable and browseable format.
  • Open uClinux Documentation Project, view/submit articles that are related to uClinux.
  • Shaolinux Temple, provides a home to documentation in progress, where visitors may contribute to its completion.


General Linux Resources:

  • Focus on Linux, has many Linux links and collections of information.
  • John's Linux Page, with Linux links, FVWM links, and general UNIX-related links.
  • Linux Fact Sheet, has summaries of information on Linux.
  • Linuxinfor, provides the latest documents covering Linux as well as other Unix-like OS's and their applications.
  • Linux Orbit, Linux desktop news, notes and recommendations.
  • Linux Places, a collection of Linux links.
  • linuxpower.org, a site dedicated to helping Linux users harness the power of Linux. Features news, links, tutorials and articles.
  • The Linux Resources.
  • Linux Start, a searchable site with information on Linux, FreeBSD and Open Source projects.
  • Linux UK, a site for UK Linux users.
  • Linux@UKUUG, has lots of information on Linux in the UK.
  • Linux VooDoo provides Linux drivers, technical support on installation, configuration and security, and games.
  • LinuxWeb has many links and information.
  • The Linux WebRing, a collection of Linux-related pages.
  • Lucc's Linux Linx, has some interesting links.
  • How-To's for Mandrake-Linux, focuses on some known services (web, mail, dns, etc) and other subjects.
  • Manfred Heyen's Homepage - Linux!, articles/information in English and German.
  • Multimedia4Linux, information about Multimedia with Linux.
  • Open ITx discussion groups, features email-based discussion groups that focus on technology.
  • Linux Razor, software, tutorials, books and more.
  • SorgoNet.com, articles: linux, hardware hacks, programming; news, help.
  • UK.LINUX.ORG, has some good general Linux information, as well as Alan Cox's projects.
  • The Web Masters Den - Resources - Linux, Many useful resources for those seeking information on Linux.
  • RHCE2B.Com, a free web site offering documentation, study guides, news and a free practice test for those people studying for the RHCE exam.
  • Linux Kernel Architecture Resources, a comprehensive listing of resources containing information about all aspects of the Linux kernel.
  • Loads of Linux Links, a GPLed database of +4,000 subject-classified and searchable Linux and Open Source links for all levels of Linux users.
  • MH Linux, Linux support forums and News portal.
  • All-Linux.net, large list of Linux resourcesi (links).
  • Programmer's Toolbox Videos; these videos are free as in speech (but not beer), meant to help people who are already programmers expand their boundaries and go further into the technology they use. Useful w/any operating system, but Linux is focused on. Videos are VideoCDs recorded on CDRs.
  • LinuxBazis, Linux links base.
  • Short Family Web Site - Bits and Bytes - Unix and Linux, a resource center for Unix and Linux administrators and users.
  • linux.startkabel.nl, portal with categorized linkas.


New Users and Help Sites:

  • Control-Escape, a site dedicated to the "Absolute Beginner", especially those migrating from Windows.
  • Leminator, Linux administration for new users, news and links.
  • Linuxdot.org, maintains the Newbie's Linux Manual, and a weekly Linux newsletter.
  • Linux Power, German site dedicated to beginners; a lot of very useful Workshops. Your first questions about Linux may be answered in our FAQ. Our Forum/Board-Members will try to help you with any problem you encounter. The official German MandrakeUser-Forum.
  • LinuxPowered.com, a help and information site for all new and experienced Linux users.
  • LinuxQuestions.org, a free Linux forum where newbies can ask questions and Linux experts can offer advice.
  • LinuxSelfhelp.com, Devoted to helping Linux users help themselves. Provides categorized, indexed and searchable help files and forums on a myriad of Linux topics.
  • tuXfiles, the Linux newbie help files. Provides tips, tutorials and links for the new Linux users, as well as some not-so-serious Linux ANTI help.
  • code-gurus.com, dedicated to the enrichment and enhancement of the Linux and *BSD operating systems.
