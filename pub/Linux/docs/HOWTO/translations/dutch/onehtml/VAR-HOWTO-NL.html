1. Over dit document

Dit is de Linux VAR HOWTO. Het bevat een opsomming van servicebedrijven die geen hardware fabriceren of softwarepakketten creëren, maar waarde toevoegen aan bestaande producten. Deze toegevoegde waarde kan alle vormen aannemen, zoals de installatie van Linux op computerhardware, de configuratie, of aangepaste applicaties. Echter op dit moment bevat de Linux VAR HOWTO alleen bedrijven die voorgeconfigureerde hardware verkopen met het doel Linux te draaien. Bedrijven die andere services bieden worden uitgenodigd contact op te nemen met de beheerder poet@linuxports.com via poets@linuxports.com voor een opsomming.

Als je contact zoekt met bedrijven die in dit document worden opgesomd, noem dan alsjeblieft de Linux VAR HOWTO.

Als je meer wilt weten over het Linux Documentation Project of Linux HOWTO's, neem dan gerust contact op met de supervisor Tim Bynum <linux-howto@sunsite.unc.edu>.

Tim Bynum zal de lijst maandelijks naar verscheidene nationale en internationale nieuwsgroepen posten. Bovendien is de Linux VAR HOWTO op het World Wide Web te vinden bij http://www.linuxports.com. Nieuwe versies van de Linux VAR HOWTO worden altijd eerst op deze site geplaatst, dus controleer of de kopie die je aan het lezen bent nog wel up to date is!

Bedrijven die hardware verkopen met als doel Linux te draaien worden uitgenodigd het formulier in te vullen dat hier te vinden is. Bedrijven die andere services aanbieden kunnen contact met me openemen zodat er een passender template kan worden gemaakt.
