8. Proyek Nasionalisasi Indonesia

Sekarang sudah cukup banyak proyek-proyek yang bertujuan untuk memudahkan akses Linux ke dalam Bahasa Indonesia. Daftar yang tidak lengkapnya (dan tidak terurut) adalah:

  • i18n Indonesia < http://i18n.linux.or.id>, koordinator Ronny Haryanto < giant@canada.com>. Tersedia pula mailing list i18n ini, untuk subscribe silahkan kirim email ke i18n-subscribe@linux.or.id
  • LDP Indonesia < http://ldp.linux.or.id>, koordinator: Mohammad DAMT < mdamt@linux.or.id>
  • KDE Indonesia < >, koordinator I Made Wiryana < mailto:made_lin@nakula.rvs.uni-bielefeld.de
  • Terjemahan man pages < >, koordinator I Made Wiryana < made_lin@nakula.rvs.uni-bielefeld.de>.
  • GNOME Indonesia, koordinator Priyadi Iman Nurcahyo < priyadi@priyadi.id.org
Dukungan dan bantuan rekan-rekan sangat diharapkan guna mensukseskan proyek-proyek ini.
