7. Perubahan-perubahan

7.1 Perubahan dari 0.3 ke 0.4

  • Penambahan lebih detil class HTML untuk memberikan kegunaan minimal.
  • Menulis class HTML_Test dan javahtmltest.html-dist.
  • Penambahan metode-metode HTML untuk menangani daftar definisi.

7.2 Perubahan-perubahan dari 0.2 ke 0.3

  • Penambahan class Text dan Email. HTML juga ditambahkan, tetapi hanya sedikit saja.
  • Menaruh berbagai class ke dalam paket. Class-class utama ada di dalam Orbits.net.*, class pendukung Text ada di dalam Orbits.text.Text.
  • Perubahan CGItest ke CGI_Test.
  • Penambahan class Email_Test

7.3 Perubahan dari 0.1 ke 0.2

  • Variabel-variabel lingkungan ditaruh ke file sementara tidak dimasukkan ke perintah baris interpreter Java. Class CGI dan java.cgi perlu dimodifikasi.
  • Dokumen javacgitest.html dijadikan bagian distribusi.
  • File-file teks yang dimodifikasi oleh make selama instalasi diberikan dengan nama-nama yang berakhiran dengan -dist.
