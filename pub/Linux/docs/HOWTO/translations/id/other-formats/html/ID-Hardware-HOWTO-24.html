24.6 Modem dan card serial

Biasanya semua card modem, card port serial biasa, dan modem digital seluler bisa dipakai. Modem ISDM yang mengemulasi standar UART juga bisa dipakai.

  • Advantech COMpad-32/85 dual serial
  • Quatech, IOTech dual RS-232 cards
  • Quatech quad RS-232 card
  • Socket Communications dual RS-232 card

24.7 Memory card

  • Intel Series 2 dan Series 2+ Flash
  • Maxtor MobileMax 16MB Flash
  • New Media SRAM
  • TDK Flash Memory SFM20W/C 20MB
