Die November 1999 LinuxFocus Ausgabe



Linux ist einen langen Weg gegangen, seit Linus Torvalds seinen ersten Kernel entwickelt hat. Seit den allerersten Anfängen war Linux ein Traum für Entwickler. Jetzt, nachdem das Betriebssystem bewiesen hat, daß es eine der besten (vielleicht DIE beste) Lösung für z.B. Netzwerk- und Internetserver bereithält, ist die Zeit reif, um die Qualität von Linux in anderen Bereichen zu testen. Um es genauer zu sagen, Bereiche, in denen traditionell nicht-Linux Plattformen wie MS Windows und Apple Mac dominierten. Die KDE (Release 2.0 erscheint demnächst) und GNOME Desktops geben Linux eine komplette und glatte GUI Umgebung. Das GUI unter Linux ist frei, offen und konfigurierbar. Software wie die verschiedenen freien Officesuites und The GIMP bringen Linux langsam in die Reichweite von nicht-technischen Computerbenutzern. Linus Torvalds weiß, was er tut, wenn er auf eine bessere USB und plug-and-play Unterstützung im nächsten Kernel (2.4) abzielt. Wie er selbst sagte, wird Linux in spätestens fünf Jahren technologisch soweit sein, die Weltvorherrschaft übernehmen zu können. Auf dem Weg dorthin muß noch eine letzte Hürde genommen werden: der Spielemarkt. Mehr neue Spiele sollten für Linux herausgebracht werden. Und das ist genau das, was momentan zu passieren beginnt...
