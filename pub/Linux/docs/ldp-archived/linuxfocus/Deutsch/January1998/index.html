Januar 1998: Erlernen der 3D Grafikprogrammierung

Diese Ausgabe von LinuxFocus präsentiert die ersten Artikel einer Serie über die Möglichkeiten der 3D Grafikprogrammierung unter Linux. Von der Installation der Bibliotheken bis zur Betrachtung unterstützter Hardware und der verschiedenen API Schnittstellen.

Software Review

  • Was ist OpenGL von Miguel Angel Sepúlveda
    Dieser Artikel stellt OpenGL kurz vor, die wohl erfolgreichste Bibliothek im Bereich professioneller 2D und 3D Grafik und startet damit auch eine Serie über die Programmierung mit OpenGL.

Hardware Review

  • 3Dfx Grafikkarten von Phil Ross
    3Dfx Interactive's 3D Chipsatz verändert die Welt der High-End 3D Grafik. Und diese Karten werden unter Linux unterstützt!
  • Linux auf einem Rechner mit Alpha CPU zum Laufen bringen von Bruce Ediger
    Der Autor schildert seine Erfahrungen beim Aufbau eines Alpha PC Systems.

Kernelecke

  • Die Geheimnisse des Kernels von Emiliano Ariel Lesende
    Dies ist eine kurze Beschreibung des Linuxkernels.

Programmierung & Entwicklung

  • GLUT: Fenster and Animationen (Teil 1) von Miguel Angel Sepúlveda
    GLUT ist das GL Utility Toolkit, entwickelt von Mark Kilgard. In diesem Artikel stellt der Autor GLUT vor und zeigterste Schritte in der Handhabung von Fenstern und Animation mit GLUT.
  • OpenGL: Rendern von einfachen Polygonen (Teil 1) von Miguel Angel Sepúlveda
    Der Autor vermittelt Grundlegendes über OpenGL Anwendungen und erläutert, wie einfache Polygone in 2D gerendert werden.
  • Code debuggen mit ddd von Jose Maria Laveda
    ddd ist ein wundervoller graphischer Debugger. In diesem Artikel werden einige grundlegenden Funktionen von ihm beschrieben.

Systemadministration

    Lerne, wie man die Verfügbarkeit von Ressourcen wie Routern, Hubs, Servern und anderen wichtigen Geräten im Netzwerk sicherstellt.
