LF 小帖子：随时改变键盘布局

Introduction

当我在电脑上写文章的时候，我会注视着屏幕而不是键盘。最近，我拜访了一个在法国的朋友，他把它的电脑借给我用了一小会儿。天哪，法国键盘！每找一个键我都得花上10秒钟，因为我习惯了美式或德式的键盘布局；-)
我可不想编辑别人电脑上的 /etc/X11/XF86Config-4文件……。KDE和Gnome自带了改变键盘布局，但这儿好像并没有安装，那该怎么办呢？  

解决办法

XFree86有一个叫setxkbmap的标准命令可以在不重启X server 的情况下改变键盘布局，这和Linux控制台下面的命令loadkey。 我输入： 
setxkbmap -layout us
马上我就有了一个美式键盘的布局了！在将电脑归还朋友前，我只需输入： 
setxkbmap -layout fr
一切如故。
