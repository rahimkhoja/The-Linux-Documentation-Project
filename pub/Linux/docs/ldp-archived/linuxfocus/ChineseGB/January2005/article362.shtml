LF Tip: LinuxFocus 活动书签

摘要:

This is a small tip. From now on LinuxFocus will have at least one new tip every month. If you have some ideas for a new tip then send them to guido(Q)linuxfocus.org

_________________ _________________ _________________

 

简介

Mozilla的Firefox如今有一个新的特性叫做“活动书签（"Live Bookmarks"）”，它可以让你的书签自动更新它的内容。当一个网页提供此特性时，Firefox会在右下角显示一个如图所视图标：  

LF 活动书签

linuxfocus.org的首页现在也提供了活动书签：

你可以通过点击一个可能的连接来给你的Firefox加入一个活动书签，譬如：可以点击"Latest LF articles"。 加入书签后，你就能在你的书签你找到最近的（翻译过的）文章： After having added an entry you can find the latest articles or the latest translated articles directly in your bookmarks folder:
