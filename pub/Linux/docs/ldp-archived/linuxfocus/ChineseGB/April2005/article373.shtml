小帖子: 解决那个丑陋的键盘问题

摘要:

This is a small tip. From now on LinuxFocus will have at least one new tip every month. If you have some ideas for a new tip then send them to guido("at" sign)linuxfocus.org

_________________ _________________ _________________

 

Introduction

你现在有了一个最好的操作系统之一,而且你也对你的桌面设置感到非常满意,那么当你看着你的电脑的时候是不是感到非常高兴于自豪呢? 然而,当你往下看到你的键盘的时候,你看到什么了? 是不是有两个丑陋的键在键盘的最下面的一行? Windows 键??!

尽管你可能并不会用到这两个键, 看到它们仍然会让人感觉到恼怒和不安. 如何能够用"企鹅键"来代替它们呢?

这篇小帖子会介绍两种解决方案.  

方案一: 一个新的键盘

德国键盘制造商 Cherry 在去年宣布他们会制造 Linux 专用的键盘, 参见他们宣告: on news.zdnet.co.uk).

在 Cherry 的网站上搜索一下你将会找到有两款比较好的Linux 键盘:
  • http://www.cherry.de/english/enjoy-line/enjoy_cymotion_master_linux.htm
  • http://www.cherry.de/english/enjoy-line/enjoy_g83-6188_linux.htm
 

方案二: 在那两个"丑键"上贴上新图标

shop.tuxgraphics.org 上有一些可贴标签,它们贴在这些键上刚好合适. 你所需要的可贴标签应该是: "tux sticker V6".

这些可贴标签与普通的标签特别之处在于: 即使你经常使用这些键, 上面的文字图像也不会脱落.

现在我的女儿非常喜欢我的键盘. 每当她看到我的键盘, 她都会面带微笑地指着这些键说:"企鹅!"
 

使用这些键

使用 xev 这个程序你可以检查你的这两个"企鹅键"是关联到哪个键码上的. 我的是键码115 和 116. 使用xmodmap 程序你可以给这些键赋予一些功能(你甚至可以将字母关联到它们上,如欧元符号). 
xmodmap -e 'keycode 115 = Super_L'
xmodmap -e 'keycode 116 = Super_R'
或者
xmodmap -e 'keycode 115 = EuroSign cent'
如何将一些窗口动作关联到这些键上则取决于你的窗口管理系统了. 如,在 gnome 里你可以使用 "桌面首选项->键盘快捷键(gnome-control-center->Keyboard Shortcuts)". 在 xfce 和 fvwm 则需要修改它们的配置文件. 注意,如果你想单独使用这些键而不想使它们与别的键组合,那么你不能将它们关联到 Super_L 和 Super_R. 因为 Super_L 和 Super_R 像Ctrl 和 Shift 一样是修饰键. 这样,你可以将它们映射到其他标记(参见 /usr/X11R6/lib/X11/xkb/symbols, 如 标记 SunOpen).  

Conclusion

没有Windows 键,而有了一些"企鹅键",生活变的美好多了 :-) !
