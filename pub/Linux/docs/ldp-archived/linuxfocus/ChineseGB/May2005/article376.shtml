Introduction

Mozilla Firefox 毫无疑问已经为万维网("world wide web")做好了准备。一些网站在提供一些有用内容的时候往往提供一些十分烦人且极不友好的广告。这里向大家推荐两个我浏览网页时常用的FireFox的扩展。
 

Flashblock

Flashblock会将所有网页中的 flash 去掉并用一个小圆按钮代替。再也没有闪烁的 flash 图像啦！如果你想看到那些 flash 动画，只要点击那个小圆按钮就可以了。

https://addons.update.mozilla.org/extensions/moreinfo.php?id=433


 

NukeAnything

Nuke Anything 能去除任何对象。你只需右击那个对象（也许甚至是段文字！），然后选择“删除对象”("remove object")，这个扩展在去除图像和任何类型的广告方面非常有用，它也可以用来在打印之前“整理”("clean up")页面。

https://addons.update.mozilla.org/extensions/moreinfo.php?id=79
