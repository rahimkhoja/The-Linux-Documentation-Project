欢迎访问LinuxFocus杂志2002年十一、十二月号


LinuxFocus杂志5岁了！他开始于1997年11月Miguel和他的朋友发布的第一期杂志。此后大量各方面的文章组成了他非常庞大的知识库，
最初的小组里，只剩下John Perr仍然和我们在一起。在1997年我还是一个使用Slackware仅仅一年多的Linux初学。
当我在1998年初发现这本杂志时，我为这种国际性杂志的创意而着迷。文章的质量没话说，而且这是一份“诚实”的杂志。
这与很多充着广告和产品介绍的商业性杂志形成鲜明的对比。我仍然记得，当时我几乎等不及下一期杂志的发布。当杂志发布后，我一直阅读到深夜。
那时，我开始自己写一些文章，当然在1999年一月号的杂志发布之后有一些变动，原来的LinuxFocus小组想放弃这份杂志，以及我成为了这份杂志的主编。。。。。。这几年来，杂志从全部用手工完成每一篇文章的编辑到现在的这个分离内容及版面的便于管理的系统。版面由脚本来生成。

LinuxFocus由新、老成员组成的小组继续着，但是直到今天他仍然保持者原有的精神。

生日快乐！

 

-- Guido
