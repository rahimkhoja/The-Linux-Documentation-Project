November 1999 LinuxFocus issue



自从Linus Torvalds开发他最初的内核以来，Linux取得了很多成就. 因为非常简明, Linux是开发者的梦想. 现在这个OS已证明它能为例如网络和国际互连网服务器提供最好解决方案之一(也许就是最好),时间正在证明Linux在其他领域的品质. 在更多专用领域,传统的非Linux平台象MS Windows和苹果Mac是主要的平台. KDE(2.0发放在即)和GNOME桌面给Linux一个完整和平滑GUI环境. Linux让GUI自由,开放和可定制. 其他软件象各种自由的办公室套件和GIMP让Linux更加接近非技术计算机用户. Linus Torvalds正在针对在下一个 (2.4)进行更好USB和plug-and-paly支持，他知道他正在做什么. 他和当年一样, Linux已经技术上准备好在最多5年内成为世界上的主流操作系统. 在这过程中,最后障碍无疑是: 游戏市场. 更多新游戏将被为 Linux发放. 这是将要立刻发生的事情的开始...
