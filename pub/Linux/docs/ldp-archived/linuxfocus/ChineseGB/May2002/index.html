欢迎访问LinuxFocus杂志2002年第三期
（五、六月号）

GNU，自由软件基金(FSF)，RedHat以及后来Linus Torvards都在美国。
人们会有一种印象，Linux和自由软件都是“made in USA”。上个月的调查 Linux around the world，显示了一些有趣的 结果。这证实了Linux和自由软件确实是一个国际化的运动。为此我们非常高兴，因为 这显示了拥有国际化的站点是多么的重要，那些告诉我们LinuxFocus杂志只需要 英文版本就足够了的人是错误的。

现在我们拥有一份大型的期刊，我没有什么其他的好说的，除了祝您阅读愉快 :-)
