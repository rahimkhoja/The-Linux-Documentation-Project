Что такое LinuxFocus?

LinuxFocus - бесплатный международный журнал для Linux. Это не коммерческая организация и поддерживается добровольцами со всего мира. LinuxFocus представляет бесплатную документацию для бесплатных ОС. LinuxFocus - международный журнал и заметки переводятся на многие языки.
