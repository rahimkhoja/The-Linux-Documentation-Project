Совет от LinuxFocus

Вас раздражают постоянные beep'ы компьютера?
  1. Отредактируем файл ~/.xinitrc. Если подобного нет в вашем домашнем каталоге - используйте системный.
    /usr/X11R6/lib/X11/xinit/xinitrc
    Скопируйте его в свой домашний каталог с именем .xinitrc и редактируйте.
  2. Добавьте строку 
    xset -b
    в начало файла
  3. После очередного запуска X-windows эти звуки пропадут.
Обратите внимание : Пользователи KDE или Gnome могут изменить значение уровня beep на ноль в центре управления (control center).
