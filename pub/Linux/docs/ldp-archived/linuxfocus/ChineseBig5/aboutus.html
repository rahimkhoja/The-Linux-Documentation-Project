關於 LinuxFocus

LinuxFocus 創刊於 1997 年十一月。一開始 Miguel Angel Sepulveda 和一些朋友感覺到非英語的 Linux 文件實在太少了，因此他們決定成立這本多國語言的雜誌。

雖然我們大多數人都會讀英文文件，書籍和新聞群組，但我們覺得， 以非英文的方式來提供資訊十分的重要。 許多在非英語系國家的年輕人所學的英文有限， 他們第一次接觸電腦的經驗很可能就是微軟的作業系統。 微軟的作業系統已經被翻譯成幾乎世界上任何一種語言的版本，而商業化的 Linux 在這方面還相差甚遠。

因此我們的理念是希望能將這種方式發揚光大，使更多人能用他們自己語言來取得資訊， 進而建立新一代的 Linux 使用者。

Miguel Angel Sepulveda 是 LinuxFocus 雜誌的創使人。他原來在 Pittsburgh 大學做聚合體和量子的研究工作。 他有一個理論化學的博士學位，且鍾愛動晝，3D 立體程式設計和模擬。 自從 1993 年成為一個 Linux 的使用者以來， 就很少有時間再聽他喜愛的古典音樂和帶小孩。 他於 1999 年辭去了 LinuxFocus 領導的職位。


LinuxFocus 的目標是出版一份免費的網上 Linux 雜誌。LinuxFocus 是一份公開的、 多語的雜誌，它是由許多人共同努力的成果，任何人也都可以投稿自己的文章。

Guido Socher 是繼 Miguel Sepulveda 離開之後 LinuxFocus 的領導人。

Guido Socher Guido 從他第一次安裝 Linux 在他的電腦上就成了一個標準的 Linux 迷。 他喜歡 Linux 不只是它的軟體，而是它的哲學。Guido 是一位電子工程的碩士， 目前在一家電信公司擔任軟體設計的工作。最近他將很多的空餘時間都花在 LinuxFocus 上。




LinuxFocus 絕對不是一個人的成果，而是集數百人的貢獻而成的。 我們還有數組負責不同語言的翻譯人手。
每一組的聯絡人可以在我們的 lf-team 網頁 上找到。

我們竭誠希望您能夠享受閱讀這份雜誌，更歡迎您的建議和批評指教。
