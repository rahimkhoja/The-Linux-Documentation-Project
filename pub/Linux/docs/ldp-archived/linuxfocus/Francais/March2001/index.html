Le "truc" de LinuxFocus

Vous détestez les bips sonores de votre ordinateur?
  1. Editez votre fichier ~/.xinitrc. S'il n'est pas présent dans votre répertoire home c'est que vous utilisez sans doute celui du système
    /usr/X11R6/lib/X11/xinit/xinitrc
    Copiez-le dans votre répertoire home sous le nom .xinitrc et éditez-le.
  2. Ajoutez la ligne 
    xset -b
    au début du fichier.
  3. Au prochain démarrage de X, votre ordinateur ne provoquera plus de "bips" intempestifs.
Note: Les utilisateurs de KDE ou Gnome devront réduire le volume à zéro dans le panneau de contrôle.
