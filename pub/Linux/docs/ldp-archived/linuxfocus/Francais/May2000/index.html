LinuxFocus Mai 2000

Tout le monde devient libre! Presque tous les éditeurs de systèmes d'exploitation sortent une version libre. Libre n'a pas toujours la même signification dans ce cas. Quand BeOS dit libre (www.be.com/products/freebeos) ça signifie gratuit. Pour QNX le Sytème Temps Réel, libre veut dire plus que gratuit. Leur OS est maintenant open source (www.qnx.com/news/pr/apr24_00-openpl.html). Chez Sun Microsystems il était déjà possible d'obtenir une copie gratuite (de solaris 7) et ils continuent en distribuant le code source de Solaris 8 (voir www.sun.com/developers/tools/solaris). Apple envisage également d'ouvrir "une petite" partie de son code (voir Darwin Open Source SDK).
Serait-ce une nouvelle manière de "vendre" du logiciel? .... ou bien serait-ce seulement une mode? Ces entreprises espèrent-elles vraiment améliorer leur code en le rendant public ou serait-ce un moyen d'attirer de nouveaux clients? Dans tous les cas cette nouvelle vague "libre" est certainement une conséquence de l'importance croissante de Linux.
