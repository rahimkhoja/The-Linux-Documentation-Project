Truc LF: Les marque-pages dynamiques sous LinuxFocus

Résumé:

C'est une petite astuce. A partir de maintenant, LinuxFocus aura au moins une nouvelle astuce tous les mois. Si vous avez des idées pour une nouvelle astuce, envoyez-les à guido(Q)linuxfocus.org

Introduction

Mozilla Firefox a une nouvelle fonctionnalité appelée « marque-pages dynamiques » (« Live Bookmarks »); elle vous permet d'avoir un dossier dans vos favoris qui met à jour automatiquement son contenu. Firefox montre une petite icône orange dans le coin inférieur droit d'une page web qui offre cette fonctionnalité.  

Les marque-pages dynamiques de Linuxfocus

La page d'accueil de linuxfocus.org offre aussi maintenant des marque-pages dynamiques :

Vous pouvez ajouter une entrée dans votre dossier des favoris de Mozilla Firefox en cliquant sur un des choix possibles, par exemple « Latest LF articles » (les derniers articles de Linux Focus). Après avoir ajouté une entrée, vous pouvez trouver les derniers articles ou les derniers articles traduits directement dans votre répertoire de favoris :
