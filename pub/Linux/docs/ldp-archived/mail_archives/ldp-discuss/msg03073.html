The LinuxDoc-Tools 0.6.1 can be downloaded from
 <http://www.debian.org/~sano/linuxdoc-tools/>

Using this, single html output with the generated toc can be produced
(man page is not udpate yet). New option "-T" is added on sgml2html 
(or linuxdoc -B html) in order to control the level of toc for html.

Try using "linuxdoc -B html -s 0 -T 2 <some linuxdoc sgml source>".

guide.sgml is updated also.
