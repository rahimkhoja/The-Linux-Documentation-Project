I've added a link on the sorted_howtos.html page to allow you
to grab the data file. This data file is in a tab-delimited (0x09)
format, with the fields being as follows:

    DATE   (YYYY-MM-DD)
    TYPE   (HOWTO/mini/unmaintained)
    TITLE
    URL    (encoded as an ftp-style link to metalab.unc.edu)

This data file will get propagated to the mirror sites. It's
extremely small (~5k; gzipped).

Sorted Howto's page:      www.linuxdoc.org/sorted_howtos.html
Sorted Howto's data file: www.linuxdoc.org/howto.d/sorted_howtos.dat.gz
