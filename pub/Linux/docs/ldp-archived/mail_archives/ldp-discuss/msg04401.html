Miroslav Skoric wrote:
> 
> Gentle folks,
> 
> I wonder if there is a complete listing of DocBook Tags used in SGML?
> Those listed in LDP Author Guide are not enough for me (for example,
> using of <bf>command_name</bf> or using of
> <command>command_name</command> looks to me to give a same boldface HTML
> result). Any solution?

See DocBook: The Definitive Guide for a complete list of DocBook tags.

http://www.docbook.org/tdg/html/docbook.html

However, the "problem" you are describing is normal. DocBook doesn't
necessarily guarantee any particular visual output. Many tags are
semantic only.
