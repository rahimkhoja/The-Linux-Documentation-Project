Does this fit somewhere into the various cataloguing and indexing
efforts for LDP and related material?

It is Perl code to implement W3C's RDF:

> The Resource Description Framework (RDF) integrates a variety of
> web-based metadata activities including sitemaps, content ratings,
> stream channel definitions, search engine data collection (web
> crawling), digital library collections, and distributed authoring,
> using XML as an interchange syntax.

RDF in turn leads to the Semantic Web. see:
http://www.w3.org/DesignIssues/
