Hello,

I have set my programmer loose on a small project that has seen its first
Alpha release today. The product can be retrieved from this url:

http://stage.linuxports.com/riptags-0.1.8.tar.gz

What this product does it take a configuration file and rip through any
file and replace it with different tags.

For example, if you put the following:


BODY	MIND


it will replace all instances of BODY with MIND. You can speicify multiple
tags. THerefore we could rip through large SGML docs and do tag
transformation for things like DocBOOK.

It will not solve all translation issues but it will make the drop a
little faster. Take a look and let us know what you think.
