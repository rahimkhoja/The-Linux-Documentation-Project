The LinuxDoc-Tools 0.7.1 can be downloaded from

 http://www.debian.org/~sano/linuxdoc-tools/ now.

(It will be available from 
   http://www.linux.or.jp/JF/workshop/archives/linuxdoc-tools.tar.gz
in a few days.)

Added the support for Korean (-l ko -c euc-kr) and the new option
to use hlatexp or platex for latex processing using the patch from
CHOI Junho. 

And fix some problems using the patch from  Evgeny Stambulchik.

The spec file for RPM written by Hayakawa and Takei for Kondara Linux
is included.

And the manual page for linuxdoc command is updated to include the
description for various backend drivers.

Please try it if you need the most bug-fixed tool for Linuxdoc DTD SGML.
