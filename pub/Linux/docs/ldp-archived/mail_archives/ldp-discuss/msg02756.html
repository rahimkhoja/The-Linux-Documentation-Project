Am 11. Jun, 2000 schwäzte M. Leo Cooper so:

> IMHO, any assistance in converting HOWTOs from text to Docbook should result
> in some sort of co-authorship of the HOWTO, or at least some mention. Has
> there been any discussion of this by the group?

I think there are several roles that can assist with a HOWTO:

author
maintainer/author
editor
marker upper (don't know what that'd be called)

The last one would probably usually end up being an editor as well, but
not always.

I think the intro section should list each of the above. A credits section
at the end could mention past maintainers, etc.
