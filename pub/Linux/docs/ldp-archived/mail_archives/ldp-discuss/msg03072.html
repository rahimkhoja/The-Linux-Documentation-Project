Stephane Bortzmeyer wrote a nice manual for Debian users.
Check his "The Debian SGML/XML HOWTO" at
 <http://www.debian.org/~bortz/SGML-HOWTO/> if you use Debian.

For potato (coming 2.2), you can install all the required packages
for DocBook SGML processing by installing "task-sgml" meta-package.

For slink (current release), you can install the required packages
by 

         apt-get install docbook docbook-doc sp jade \
            docbook-stylesheets jadetex debiandoc-sgml \
            psgml
