I agree with you that in general, HOWTO's should not be distribution-specific.
Unfortunately, the creation of any root filesystem will be dependent on the
distribution, i.e. a SUSE root filesystem is installed differently than a
Mandrake root filesystem.  Any documentation written on this subject will have
to reflect this difference.
