    David> Personally I can't quite determine what the "hoo-ha" about

XML has veryinteresting issues. And it seems to be the next huge
standard. There are lots of applications that can generate XML
output. 

One big advantage of XML is to have a self-contained document. It has
the element's definition, the stylesheets and the document itself. Of
course these three could be separated in three different documents,
but the fact that it can be all together is very very nice for people
doing documents. 

I'm starting to study it to implement XML at Conectiva... By now we're
using SGML and DSSSL stylesheets. Due to Jade's DSSSL limitations, I'm
doing some customization on the backend too... With XML I won't need
to do that.
