Keywords and index terms ... someday we may want one LDP doc to
cross-reference another.  That leads to policies on first and
second-level index terms.

Other policies are the format for Bibliographic entries and for the
style used for XREF tags, for example, I carry over my LaTeX
convention of using xref tags of the form TYPE:TAG (ie FIG:NETWORK vs
TAB:NETWORK) only DocBook does not allow the colon as an ID character.
The idea is to make such anchors easy to guess from the context and
to define a sub-namespace for each type of cross-reference (ie SEC vs
FIG vs TAB &c)
