LDP SGML/DSSSL system, so you should be able to do your work in XML
and submit the files to the LDP without changes.

The reverse is not true: There are constructs possible in SGML which
are not supported in XML (such as using the IGNORE/INCLUDE directives
to tell Jade which parts to skip and which to print).
