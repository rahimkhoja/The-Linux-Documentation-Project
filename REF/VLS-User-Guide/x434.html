3.2. Administration interface

There are currently two ways to launch the streaming :

  • you can use the command line to give arguments at startup;

  • or you can use the telnet interface to start/stop/pause the streaming whenever you want .

When using the telnet interface, you must authenticate before typing any command, because any user may not be allowed to execute any command (this can be configured in the vls.cfg configuration file) .
